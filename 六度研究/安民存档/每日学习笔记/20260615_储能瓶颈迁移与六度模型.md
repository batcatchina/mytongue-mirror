# 20260615_储能瓶颈迁移与六度模型

> 来源：新华网/储能行业资讯汇总
> 日期：2026-06-15
> 主题：储能行业最新动态与六度投资模型对接

---

## 核心观点（5条）

1. **瓶颈迁移**：储能从"可选"变"必选"，十五五投资超5万亿，年均储能投资较2022年增长近8倍，成为电力系统核心瓶颈

2. **锚链盈利**：114号容量电价落地，IRR稳定6%-12%，独立储能盈利模式从"靠天吃饭"转向"固定+浮动"双收益

3. **稀缺迁移**：AI算力配套储能需求爆发，占比突破40%超越传统风光配储，1GW算力年耗电7000GWh，储能成为刚需

4. **反者道之动**：电芯价格从0.29→0.35元/Wh（+20%），4个月排产新高，价格战结束，龙头议价能力提升

5. **二线alpha**：液流/钠离子等二线技术路线受益于政策定向支持，在细分赛道中可能跑出超额收益

---

## 六度模型对接

| 六度维度 | 对接内容 |
|---------|---------|
| **瓶颈迁移** | 储能从电力系统补充角色→核心支撑要素；新型电网投资瓶颈向储能迁移 |
| **锚链盈利** | 容量电价165-330元/kW·年形成稳定收益锚；IRR 6%-12%提供安全垫 |
| **稀缺迁移** | AI算力储能从配套→稀缺资源；稀缺从"发多少电"转向"存多少电" |
| **反者道之动** | 价格跌至极低→触底反弹；内卷见底→供需再平衡 |
| **二线alpha** | 液流/钠离子等二线技术路线，政策定向支持，高增长低估值 |

## 投资启示

- **底仓配置**：独立储能运营商（IRR 6%-12%，稳定收益）
- **进攻配置**：AI算力配套储能（需求增速最快，政策加持）
- **弹性配置**：液流/钠离子二线技术（政策定向，低估值高增长）
- **周期博弈**：电芯涨价周期中的龙头电池企业（宁德时代、比亚迪）
