## 核心规则详解

### ⚠️ P0-产后-三特征基础判断

**规则逻辑**：
```
IF 产后（1月内）
THEN 舌象基础判断：
  - 舌质淡白 → 失血耗气（正常生理期）
  - 苔薄易剥 → 气血不足（正常生理期）
  - 裂纹舌 → 阴血亏虚（需调理）
```

### ⚠️ P0-产后-艾灸禁忌

**规则逻辑**：
```
IF 产后恶露未尽
THEN 禁灸：关元、气海、归来等下腹穴位
IF 产后伤口未愈
THEN 避开：伤口局部及周围
```
