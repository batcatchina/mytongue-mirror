## 核心规则详解

### P1-体质辨识四步法

```
Step 1：望舌色
  → 淡白 → 气虚/阳虚/血虚
  → 淡红 → 平和质（正常）
  → 红/绛 → 阴虚/热证/湿热
  → 紫暗 → 血瘀/寒凝

Step 2：望舌苔
  → 薄白 → 正常/表证
  → 白厚 → 痰湿/寒湿/积食
  → 黄苔 → 热证/湿热
  → 少/剥 → 阴虚/气阴两虚

Step 3：望舌形
  → 胖大齿痕 → 气虚/阳虚/痰湿
  → 瘦薄 → 阴虚/血虚
  → 裂纹 → 阴虚/热盛/血虚
  → 瘀点 → 血瘀

Step 4：综合判定
  → 舌色 + 苔色 + 舌形 = 体质类型
```

### P1-九种体质舌象速查

| 体质 | 舌质 | 舌苔 | 舌形 |
|------|------|------|------|
| 平和质 | 淡红 | 薄白 | 正常 |
| 气虚质 | 淡白 | 薄白 | 胖大齿痕 |
| 阳虚质 | 淡白 | 白滑 | 胖大齿痕 |
| 阴虚质 | 红 | 少苔 | 裂纹 |
| 痰湿质 | 淡胖 | 白厚腻 | 胖大 |
| 湿热质 | 红 | 黄腻 | 可胖大 |
| 血瘀质 | 紫暗 | 少苔 | 瘀点 |
| 气郁质 | 红/暗红 | 薄白 | 可偏瘦 |

### P1-复合体质识别

```
体质转化规律：
  阳虚 → 痰湿（阳虚水泛，湿聚成痰）
  阴虚 → 血瘀（阴虚内热，热灼血液）
  气虚 → 血瘀（气为血帅，气虚血瘀）
  痰湿 → 湿热（郁久化热）
```

### P1-体质问诊必问项

```
气虚质：疲劳否？易感冒否？声低弱否？
阳虚质：怕冷否？手脚凉否？便溏否？
阴虚质：口干否？手心热否？盗汗否？
痰湿质：身重否？痰多否？体胖否？
湿热质：油光否？口苦否？便黏否？
血瘀质：刺痛否？固定否？晦暗否？
```
