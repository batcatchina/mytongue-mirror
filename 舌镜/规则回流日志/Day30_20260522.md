## 核心规则详解

### ⚠️ P0-儿童-白厚苔辨证规则

**规则逻辑**：
```
IF 儿童 AND 舌苔白厚
THEN 区分病因
  - 苔腻+口臭+腹胀+便酸臭→积食→消食导滞
  - 怕冷+手脚凉+便稀+不爱动→虚寒→温中健脾
  - 面色潮红+口干→积食化热→清热消积
问诊必问：食欲、大便性状、腹部胀满否
```

### ⚠️ P0-特殊人群-就医红线规则

**规则逻辑**：
```
IF 儿童 AND (草莓舌+高烧+皮疹+淋巴结肿大) OR (地图舌>3月+体重不增) OR (舌体肿大发紫+呼吸困难)
THEN 立即建议就医
  - 不可自行调理
  - 需排查感染/心肺问题
```

### P1-儿童-舌象四特征

| 舌象 | 病机 | 处理原则 |
|------|------|----------|
| 白厚苔 | 积食/虚寒 | 消食/温中 |
| 草莓舌 | 实热/虚热/感染 | 区分实虚，警惕感染 |
| 地图舌 | 脾胃气阴不足 | 养脾胃 |
| 舌底青筋粗黑 | 循环偏弱 | 排查心肺功能 |

### P1-老年人-配穴补虚原则

**规则逻辑**：
```
IF 老年人(>60岁) AND 需针灸调理
THEN 补虚为主
  - 气虚：足三里+关元+百会
  - 阳虚：命门+肾俞+关元（灸）
  - 阴虚：太溪+三阴交+照海
  - 血瘀：血海+膈俞（轻刺激）
原则：轻柔、补法为主、宁少勿多
```

### P1-慢性病-舌象对照

| 病种 | 舌象 | 提示 |
|------|------|------|
| 高血压 | 舌红少苔/苔黄 | 肝阳上亢 |
| 糖尿病 | 舌红少津/裂纹 | 阴虚燥热 |
| 冠心病 | 舌紫暗/瘀斑 | 心脉痹阻 |
| 慢性胃炎 | 舌淡苔白/厚腻 | 脾胃虚弱 |
| 慢支哮喘 | 舌淡苔白/水滑 | 肺肾气虚 |
