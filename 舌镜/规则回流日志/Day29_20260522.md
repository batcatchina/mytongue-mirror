## 核心规则详解

### ⚠️ P0-产后-恶露不绝辨证规则

**规则逻辑**：
```
IF 产后（<6周）AND 恶露量多/少/淋漓不尽
THEN 产后恶露不绝
ACTION:
  - 恶露色淡红质清稀→气虚证→补中益气汤+阿胶、艾叶、乌贼骨
  - 恶露色深红质稠→血热证→保阴煎
  - 恶露色紫暗有块→血瘀证→生化汤+失笑散
问诊：必问恶露量、色、质、味、小腹疼痛、全身症状
```

### P1-产后-身痛配穴规则

**规则逻辑**：
```
IF 产后身痛 AND 辨证分型
THEN 配穴组合
  - 血虚：足三里+三阴交+膈俞
  - 风寒：命门+关元+局部穴
  - 肾虚：肾俞+太溪+命门
  - 血瘀：血海+膈俞+局部穴
```

### P1-产后-三禁原则

**规则逻辑**：
```
IF 产后（<6周）AND 需针灸调理
THEN 严格遵守三禁
  - 禁大汗（伤津）
  - 禁峻下（伤正气）
  - 禁通利小便（伤肾气）
```
