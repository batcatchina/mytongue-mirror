## 一、经脉循行

- **起点**：手食指桡侧端（商阳穴）
- **上行路线**：沿食指桡侧缘，经第1、2掌骨间（合谷穴），进入两筋之间，沿前臂桡侧上行
- **肘部**：至肘外侧（曲池穴）
- **上臂**：沿上臂外侧前缘，上肩，出肩峰前缘（肩髃穴）
- **背部**：向后与督脉大椎穴交会
- **缺盆**：向前下入缺盆（锁骨上窝）
- **体内支脉1（入胸）**：络肺，下膈，属大肠
- **体内支脉2（上头）**：从缺盆上走颈部，经面颊，入下齿龈，回绕至上唇，交叉于人中，左脉向右，右脉向左，止于对侧鼻翼旁（迎香穴），与足阳明胃经相接

## 三、生理功能

- **传导糟粕**：主司传送、排泄食物残渣，为"传导之官"
- **主津液**：吸收食物残渣中的部分水分，参与调节体内水液代谢
- **通降为顺**：以通为用，以降为顺，保持大便通畅
- **与肺相表里**：肺气肃降有助于大肠传导，大肠传导正常有助于肺气宣发肃降

## 四、辨证规则

### 规则D39-1：大肠湿热证
- **条件**：舌质红，苔黄厚腻，脉滑数；伴有腹痛、里急后重、下痢脓血、肛门灼热、小便短赤
- **结论**：大肠湿热内蕴，气机阻滞
- **行动**：清热利湿，调理大肠。配穴：合谷、曲池、上巨虚、阴陵泉
- **⚠️ P0**：禁用温补固涩之品（如诃子、肉豆蔻），以免闭门留寇

### 规则D39-2：大肠津亏证
- **条件**：舌红少津或有裂纹，苔薄黄或燥，脉细数或细涩；伴有大便干结如羊屎、数日一行、口干咽燥
- **结论**：津液不足，肠道失于濡润
- **行动**：增液润燥，滑利润肠。配穴：天枢、支沟、照海、上巨虚
- **⚠️ P0**：忌用攻下峻剂，慎用苦寒伤津之品

### 规则D39-3：大肠虚寒证
- **条件**：舌淡胖，苔白滑，脉沉迟或细弱；伴有腹痛喜温喜按、肠鸣、大便溏薄、四肢不温
- **结论**：阳气不足，大肠传导无力
- **行动**：温阳散寒，涩肠止泻。配穴：神阙、天枢、关元、足三里
- **⚠️ P0**：高龄体虚者慎用强刺激，孕妇禁用合谷、三阴交

### 规则D39-4：大肠气滞证
- **条件**：舌淡红或暗红，苔薄白或薄黄，脉弦；伴有腹胀腹痛（痛无定处、排气后减轻）、便秘或便溏不爽、嗳气
- **结论**：气机郁滞，大肠通降失常
- **行动**：理气导滞，疏通大肠。配穴：合谷、曲池、天枢、大横、行间
- **⚠️ P0**：孕妇慎用合谷，婴幼儿便秘慎用攻下

## 八、问诊要点（十问歌对应）

| 十问 | 大肠经对应 |
|------|-----------|
| 一问寒热 | 有无发热、恶寒？是否伴有汗出异常？ |
| 二问汗 | 自汗/盗汗/无汗？汗出与排便关系？ |
| 三问头身 | 头痛（阳明经头痛在额面部）、肩臂外侧疼痛？ |
| 四问便 | 大便频率、性状（干结/溏薄/脓血）、颜色、气味、里急后重、肛门灼热？ |
| 五问饮食 | 食欲、口干、口臭、口苦、咽干？ |
| 六问胸 | 腹胀（阳明腑气不降）、腹痛喜按拒按？ |
| 七聋 | 面部/牙部/咽喉感觉异常？ |
| 八渴 | 口渴与否，喜饮冷还是热饮？ |
| 九问旧病 | 慢性肠炎、痔疮、便秘史？ |
| 十问因 | 饮食不节（辛辣/生冷）、情志波动、劳累？ |

## 九、禁忌与注意事项

### ⚠️ P0（安全禁忌）
- 合谷穴孕妇慎用（可致流产）
- 大肠湿热证禁用温补固涩之品
- 大肠津亏证慎用苦寒攻下
- 高龄体虚者慎用强刺激

### P1（重要）
- 肩髃穴针刺不宜过深，避免伤及肺尖
- 实证泻法为主，虚证补法为主
- 保持大便通畅，养成定时排便习惯
