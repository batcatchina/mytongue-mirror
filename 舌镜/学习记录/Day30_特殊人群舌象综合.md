## 一、儿童舌象特点

### 1.1 儿童生理特点
- **脾胃功能尚未健全**：消化吸收能力弱
- **阴阳稚嫩**：易虚易实、易寒易热
- **纯阳之体**：生机旺盛，恢复力强

### 1.2 儿童正常舌象
- 舌质：淡红
- 舌苔：薄白
- 特点：舌体偏小，舌苔易变

### 1.3 儿童异常舌象辨析

| 舌象 | 可能问题 | 鉴别要点 | 处理原则 |
|------|----------|----------|----------|
| **白厚苔** | 积食/虚寒 | 苔腻口臭=积食；怕冷便稀=虚寒 | 积食要消，虚寒要温 |
| **草莓舌** | 积食化热/阴虚/感染 | 便秘臭=实火；盗汗便干=虚火；高烧皮疹=感染 | 区分实虚，警惕感染 |
| **地图舌** | 脾胃气阴不足 | 90%非缺营养 | 清淡饮食、养脾胃 |
| **舌底青筋粗黑** | 循环偏弱 | 伴唇紫/气喘→警惕心肺问题 | 排查器质性病变 |

### 1.4 儿童问诊要点（十问歌精简版）

```
一问饮食（食欲、偏好）
二问大便（性状、频次）
三问睡眠（安稳/哭闹）
四问出汗（白天/夜间）
五问情绪（活泼/倦怠）
```

### 1.5 儿童舌诊安全配穴

| 症状 | 配穴 | 注意事项 |
|------|------|----------|
| 积食 | 足三里、中脘 | 轻柔点按，不宜针刺 |
| 腹泻 | 天枢、关元 | 艾灸温和灸 |
| 便秘 | 摩腹、支沟 | 食疗优先 |
| 盗汗 | 太溪、复溜 | 滋阴为主 |

### 1.6 儿童舌诊红线 ⚠️
出现以下情况务必就医：
- 持续草莓舌 + 高烧/皮疹/淋巴结肿大
- 地图舌持续3个月 + 体重不增
- 舌体肿大发紫 + 呼吸困难
- 舌面溃疡 + 无法进食

## 二、老年人舌象特点

### 2.1 老年人生理特点
- **阳气渐衰**：肾精亏虚
- **脾胃虚弱**：运化无力
- **气血不足**：阴阳失调

### 2.2 老年人正常舌象
- 舌质：淡红或偏淡
- 舌苔：薄白或白滑
- 特点：舌体偏胖，边有齿痕（生理性）

### 2.3 老年人异常舌象辨析

| 舌象 | 病机 | 辨证要点 |
|------|------|----------|
| 舌淡苔白滑 | 阳虚水泛 | 畏寒肢冷、腰膝酸软 |
| 舌红少苔 | 阴虚火旺 | 潮热盗汗、口干咽燥 |
| 舌胖大齿痕 | 脾虚湿盛 | 神疲乏力、大便溏薄 |
| 舌紫暗有瘀斑 | 血瘀证 | 刺痛固定、情志抑郁 |
| 苔黄腻 | 湿热蕴结 | 口苦黏腻、大便黏滞 |

### 2.4 老年人舌诊配穴原则

**核心原则**：补虚为主，顾护正气

| 证型 | 主穴 | 配穴 |
|------|------|------|
| 气虚 | 足三里、关元 | 百会、气海 |
| 阳虚 | 命门、肾俞 | 关元、灸法 |
| 阴虚 | 太溪、三阴交 | 照海、滋阴 |
| 血瘀 | 血海、膈俞 | 局部取穴 |
| 痰湿 | 阴陵泉、丰隆 | 中脘、健脾 |

### 2.5 老年人问诊要点

```
一问寒热二问汗（畏寒/潮热）
三问头身四问便（大便通畅否）
五问饮食六问眠（食欲、睡眠）
七问情志八问痛（疼痛部位）
九问既往十用药（慢病、药物）
```

## 三、慢性病患者舌象特点

### 3.1 慢性病舌象共性特征
- **舌质**：多淡（气血虚）或暗红（久病入络）
- **舌苔**：多厚腻（痰湿）或花剥（阴伤）
- **舌下脉络**：多曲张紫暗（血瘀）

### 3.2 常见慢性病舌象

| 病种 | 典型舌象 | 辨证要点 |
|------|----------|----------|
| **高血压** | 舌红少苔/苔黄 | 肝阳上亢/阴虚阳亢 |
| **糖尿病** | 舌红少津/裂纹 | 阴虚燥热 |
| **冠心病** | 舌紫暗/瘀斑 | 心脉痹阻/血瘀 |
| **慢性胃炎** | 舌淡苔白/厚腻 | 脾胃虚弱/湿浊中阻 |
| **慢支哮喘** | 舌淡苔白/水滑 | 肺肾气虚/痰饮伏肺 |
| **慢性肾病** | 舌淡胖苔白/水滑 | 脾肾阳虚/水湿泛滥 |

### 3.3 慢性病问诊要点

```
一问病程（多久了）
二问控制（指标稳定否）
三问用药（中西药）
四问症状（主诉+伴随）
五问生活（饮食睡眠情志）
```

### 3.4 慢性病配穴注意事项

⚠️ **安全第一原则**：
1. **长期用药者**：注意针灸与药物相互作用
2. **凝血障碍者**：禁用活血穴位，慎用针刺
3. **心脏起搏器**：禁用高频电针
4. **骨质疏松者**：避开脊柱穴位，轻柔操作
5. **糖尿病患者**：注意皮肤感染风险

## 四、特殊人群辨证要点汇总

### 4.1 年龄分组配穴策略

| 年龄段 | 核心特点 | 配穴原则 |
|--------|----------|----------|
| **儿童（<12岁）** | 脾胃弱、变化快 | 轻柔、摩腹为主、慎针 |
| **青年（12-40岁）** | 气血充盛 | 平补平泻、调理为主 |
| **中年（40-60岁）** | 虚实夹杂 | 补泻兼施、兼顾情志 |
| **老年（>60岁）** | 阳气渐衰 | 补虚为主、顾护正气 |

### 4.2 特殊人群安全穴位清单

**儿童安全穴位**：足三里、中脘、天枢（点按/艾灸）
**孕妇安全穴位**：内关、神门、三阴交（远端）
**老年人安全穴位**：足三里、关元、命门（补法）
**慢性病安全穴位**：四肢远端穴位为主

### 4.3 禁忌穴位速查

| 人群 | 绝对禁忌 | 相对禁忌 |
|------|----------|----------|
| 儿童 | 面部危险区 | 脊柱穴位 |
| 孕妇 | 合谷、肩井、三阴交 | 腹部、腰骶 |
| 老年人 | - | 强刺激、破气穴位 |
| 慢性病出血倾向 | 活血穴位 | 针刺放血 |

## 五、问诊问题生成规则（特殊人群专项）

### 5.1 儿童问诊问题生成

根据儿童舌象特征，优先生成以下问题：
```
舌象：白厚苔 + 齿痕
追问：孩子平时食欲如何？大便酸臭吗？
→ 区分积食 vs 虚寒

舌象：草莓舌
追问：孩子最近发烧吗？皮疹有吗？
→ 区分实热 vs 感染

舌象：地图舌
追问：孩子容易累吗？出汗多吗？
→ 气阴两虚评估
```

### 5.2 老年人问诊问题生成

```
舌象：舌淡苔白滑
追问：怕冷还是怕热？腰膝酸软吗？
→ 阳虚评估

舌象：舌红少苔
追问：手心脚心热吗？晚上盗汗吗？
→ 阴虚火旺评估

舌象：舌紫暗有瘀斑
追问：哪里刺痛？活动后加重吗？
→ 血瘀证确认
```

### 5.3 慢性病问诊问题生成

```
舌象：舌淡苔白 + 裂纹
追问：口干喝水能缓解吗？血糖控制好吗？
→ 糖尿病辨证

舌象：舌紫暗
追问：胸口闷痛吗？后背放射吗？
→ 冠心病评估
```
