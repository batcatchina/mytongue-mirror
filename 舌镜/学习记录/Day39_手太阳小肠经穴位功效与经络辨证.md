## 一、经脉循行

手太阳小肠经起于手小指尺侧端（少泽），沿手背、上肢外侧后缘上行，绕行肩胛部，交于大椎，前行入缺盆，络心，沿食管下行，过膈肌，抵胃，属小肠。其支脉从缺盆上行，经颈侧、面颊，至目外眦，转入耳中（听宫）。另一支脉从面颊分出，上行至目内眦（睛明），与足太阳膀胱经相接。

## 三、辨证规则

### 规则D39-1：心火下移小肠证
- **条件**：舌质红，舌尖尤甚，苔薄黄 + 心烦口渴，口舌生疮，小便短赤涩痛，或尿血
- **结论**：心火下移小肠证
- **行动**：清心泻火，导热下行。配穴：少泽、前谷、腕骨、少府（心经）、通里（心经）、照海（肾经）
- **⚠️ P0**：孕妇禁用少泽、腕骨

### 规则D39-2：小肠实热证
- **条件**：舌质红，苔黄腻 + 小便赤涩灼痛，或尿血，心烦，口疮，咽痛，腹胀，便秘
- **结论**：小肠实热证
- **行动**：清热利湿，通利小肠。配穴：腕骨、阳谷、小海、下巨虚（胃经）、阴陵泉（脾经）
- **⚠️ P0**：实热证忌用灸法

### 规则D39-3：小肠虚寒证
- **条件**：舌质淡胖，苔白滑 + 小腹隐痛喜按，肠鸣泄泻，完谷不化，小便清长，神疲乏力，畏寒肢冷
- **结论**：小肠虚寒证
- **行动**：温阳散寒，健脾止泻。配穴：后溪、养老、关元（任脉）、气海（任脉）、足三里（胃经）、脾俞（膀胱经）
- **⚠️ P0**：孕妇禁用后溪、养老

### 规则D39-4：饮停肠间证
- **条件**：舌质淡暗，苔白滑或水滑 + 肠鸣漉漉，腹胀如鼓，按之有水声，头晕目眩，心悸气短，小便不利
- **结论**：饮停肠间证
- **行动**：温阳化饮，利水渗湿。配穴：支正、小海、水分（任脉）、阴陵泉（脾经）、三阴交（脾经）、中极（任脉）
- **⚠️ P0**：孕妇禁用中极

## 四、舌象特征总结

| 证型 | 舌质 | 舌苔 | 关键特征 |
| :--- | :--- | :--- | :--- |
| **心火下移小肠** | 红，舌尖尤甚 | 薄黄 | 舌尖红赤为关键 |
| **小肠实热** | 红 | 黄腻 | 苔黄腻，小便异常 |
| **小肠虚寒** | 淡胖 | 白滑 | 舌淡胖，苔白滑，伴虚寒象 |
| **饮停肠间** | 淡暗 | 白滑或水滑 | 舌淡暗，苔水滑，肠鸣水声 |

## 五、与其他经脉鉴别

| 经脉 | 主要病候 | 核心鉴别点 |
| :--- | :--- | :--- |
| **手太阳小肠经** | 消化吸收障碍（泄泻、腹胀）、泌尿系统症状（小便赤涩）、头面五官病（耳聋、目黄、咽喉痛）、肩臂后侧痛。 | 核心在"泌别清浊"，症状多与"水液代谢"和"消化吸收"相关。 |
| **手少阳三焦经** | 水液代谢障碍（水肿、小便不利）、气机不畅（腹胀、胁痛）、头侧五官病（偏头痛、耳聋、目锐眦痛）、上肢外侧中线痛。 | 核心在"通调水道"与"气机"，症状多与"水液运行"和"气郁"相关。 |
| **足阳明胃经** | 消化系统病（胃痛、呕吐、腹胀、便秘）、头面病（前额痛、口眼歪斜、齿痛）、下肢前侧痛、热病。 | 核心在"受纳腐熟"，症状多与"胃气不降"和"阳明热盛"相关，疼痛多在前额、下肢前侧。 |

## 六、十问歌问诊要点

| 问诊项 | 小肠经相关要点 |
| :--- | :--- |
| **一问寒热** | 有无畏寒（虚寒）或发热（实热） |
| **二问汗** | 有无自汗（虚）或盗汗（阴虚） |
| **三问头身** | 有无头痛（后枕、颈项）、肩臂后侧痛、腰背痛 |
| **四问便** | 小便：是否短赤、涩痛、清长、频数；大便：是否泄泻、完谷不化、便秘 |
| **五问饮食** | 有无食欲不振、腹胀、口渴、口疮 |
| **六问胸腹** | 有无心烦、胸闷、小腹胀痛、肠鸣 |
| **七问聋** | 有无耳鸣、耳聋、听力下降 |
| **八问渴** | 是否口渴喜冷饮（实热）或不渴（虚寒） |
| **九问旧病** | 有无消化系统、泌尿系统、肩颈疾病史 |
| **十问因** | 发病诱因：饮食不节、外感寒湿、情志不畅等 |

## 七、禁忌与注意事项

### ⚠️ P0（安全禁忌）
- **⚠️ 孕妇禁忌**：孕妇禁用或少泽、后溪、腕骨、养老等具有较强通经、活血、催产作用的穴位，尤其在孕早期。如需使用，必须在专业医师指导下进行。
- **⚠️ 针刺禁忌**：天容、天窗穴深部有重要血管神经，针刺时需严格掌握角度和深度，避免伤及颈动脉、迷走神经等。颧髎穴针刺不宜过深，避免伤及面神经。
- **⚠️ 灸法禁忌**：实热证（如小肠实热、心火下移）忌用灸法，以免助热伤阴。皮肤破损、感染处禁灸。

### P1（重要）
- **辨证准确**：小肠经证型常与心、脾、肾等脏腑相关，需四诊合参，明确病位、病性。
- **配穴精当**：根据具体证型选择主穴和配穴，如虚寒证多配任脉、胃经穴位以温补；实热证多配心经、膀胱经穴位以清利。
- **手法适宜**：虚证多用补法或温灸，实证多用泻法或点刺放血（如少泽点刺放血治疗乳痈）。
- **综合调理**：小肠经疾病常与饮食不节、情志失调有关，应配合饮食调养（如清淡饮食、避免生冷油腻）和情志疏导。
- **观察反应**：治疗过程中密切观察患者舌象、症状变化，及时调整方案。
