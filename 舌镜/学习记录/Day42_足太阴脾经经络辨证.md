## 一、经脉循行与核心病候

### 1. 经脉循行路线
- **起点**：足大趾内侧端（隐白穴）
- **上行路线**：沿足内侧赤白肉际 → 内踝前缘 → 小腿内侧正中线 → 膝内侧 → 大腿内侧前缘 → 腹部
- **属络关系**：属脾，络胃
- **内部联系**：穿过膈肌 → 沿食管旁 → 连舌根，散舌下
- **支脉**：从胃分出，上过膈，注入心中

### 2. 是动则病（经脉异常表现）
- **舌本强**：舌根强硬，言语不利
- **食则呕**：进食后易呕吐
- **胃脘痛**：胃部疼痛
- **腹胀善噫**：腹胀，频繁嗳气
- **得后与气则快然如衰**：排便或排气后症状缓解
- **身体皆重**：全身沉重乏力

### 3. 是主所生病（脏腑功能异常）
- **舌本痛**：舌根疼痛
- **体不能动摇**：身体沉重，活动不利
- **食不下**：食欲不振，吞咽困难
- **烦心**：心烦，心悸
- **心下急痛**：心窝部拘急疼痛
- **溏瘕泄**：大便稀溏或痢疾
- **水闭**：小便不利
- **黄疸**：皮肤、巩膜发黄
- **不能卧**：不能平卧
- **强立股膝内肿厥**：站立时大腿、膝内侧肿胀、厥冷
- **足大趾不用**：足大趾活动不利

## 二、生理功能与辨证规则

### 1. 生理功能
- **主运化**：运化水谷精微（营养）和水湿（代谢废物）
- **主统血**：统摄血液在脉管中正常运行，防止出血
- **主升清**：将精微物质上输至心肺、头目
- **与胃相表里**：脾升胃降，燥湿相济，共同完成消化吸收

### 2. 辨证规则

#### 规则D42-1：脾气虚证
- **条件**：腹胀便溏 + 食欲不振 + 乏力少气 + 舌淡苔白 + 脉弱
- **结论**：脾气虚证
- **行动**：补脾益气。取穴：足三里、三阴交、气海、脾俞
- **⚠️ P0禁忌**：忌用泻法，忌食生冷寒凉

#### 规则D42-2：脾阳虚证
- **条件**：脘腹冷痛 + 喜温喜按 + 四肢不温 + 舌淡胖苔白滑 + 脉沉迟
- **结论**：脾阳虚证
- **行动**：温中散寒。取穴：神阙（灸）、关元、足三里、阴陵泉
- **⚠️ P0禁忌**：忌针刺过深，忌食生冷

#### 规则D42-3：脾虚湿盛
- **条件**：脘腹胀满 + 身重困倦 + 舌苔厚腻 + 脉濡缓
- **结论**：脾虚湿盛
- **行动**：健脾利湿。取穴：阴陵泉、三阴交、水分、丰隆
- **⚠️ P0禁忌**：忌滋腻碍胃之品

#### 规则D42-4：脾不统血
- **条件**：各种出血（便血、尿血、崩漏）+ 舌淡苔白 + 脉细弱
- **结论**：脾不统血
- **行动**：补脾摄血。取穴：隐白（灸）、血海、三阴交、脾俞
- **⚠️ P0禁忌**：忌用活血化瘀药

#### 规则D42-5：中气下陷
- **条件**：脱肛、子宫脱垂、胃下垂 + 舌淡苔白 + 脉虚弱
- **结论**：中气下陷
- **行动**：升阳举陷。取穴：百会、气海、足三里、提托
- **⚠️ P0禁忌**：忌久蹲、负重

## 五、十问歌对应问诊要点

| 十问 | 脾经对应 |
|------|---------|
| 一问寒热 | 脾阳虚畏寒；脾气虚无明显寒热 |
| 二问汗 | 气虚自汗；阳虚冷汗 |
| 三问头身 | 身重困倦（脾虚湿盛）；乏力少气（脾气虚） |
| 四问便 | 脾虚便溏；脾虚湿盛大便黏腻；脾阳虚五更泄 |
| 五问饮食 | **核心**：食欲不振；食后腹胀；喜热饮（阳虚）；厌油腻（湿盛） |
| 六问胸 | 脘腹胀满；心下急痛；腹部喜按（虚）/拒按（实） |
| 七聋 | 脾虚气血不能上荣可致耳鸣 |
| 八渴 | 脾虚湿盛口黏不渴；脾阳虚口淡不渴 |
| 九问睡眠 | 脾虚则卧不安；思虑过度失眠 |
| 十问因 | 饮食不节；劳倦过度；思虑伤脾；久病体虚 |

## 六、与其他经脉鉴别

| 鉴别点 | 脾经 | 胃经 | 心经 |
|--------|------|------|------|
| 病位 | 腹部为主 | 胃脘为主 | 心胸部 |
| 主要症状 | 腹胀便溏、乏力、出血 | 胃痛、呕吐、消谷善饥 | 心悸失眠、胸痛 |

| 关键穴位 | 三阴交、阴陵泉、隐白 | 足三里、内庭、中脘 | 神门、通里 |

## 七、禁忌与注意事项

### ⚠️ P0（安全禁忌）
- **三阴交、血海**：孕妇慎用
- **腹部穴位（大横、腹结等）**：需排空膀胱，避免刺伤内脏
- **脾气虚/脾阳虚**：忌食生冷、寒凉、油腻
- **脾不统血**：忌用活血化瘀药
- **中气下陷**：忌久蹲、久站、负重

### P1（重要）
- 隐白浅刺或灸法，避免化脓
- 崩漏、月经过多应排除器质性病变
- 脾虚久泻需排除感染性肠病
