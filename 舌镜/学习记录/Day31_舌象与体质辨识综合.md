## 一、中医体质分类概述

### 1.1 九种基本体质（中华中医药学会标准）

| 体质类型 | 核心特征 | 舌象特征 |
|----------|----------|----------|
| **平和质** | 阴阳气血调和，体态适中 | 舌淡红，苔薄白 |
| **气虚质** | 元气不足，疲乏气短 | 舌淡胖，苔白，齿痕 |
| **阳虚质** | 阳气不足，畏寒怕冷 | 舌淡胖，苔白滑，水滑 |
| **阴虚质** | 阴液亏少，口干咽燥 | 舌红少津，少苔，裂纹 |
| **痰湿质** | 痰湿凝聚，体形肥胖 | 舌胖大，苔厚腻 |
| **湿热质** | 湿热内蕴，面垢油光 | 舌红苔黄腻 |
| **血瘀质** | 血行不畅，肤色晦暗 | 舌紫暗，有瘀点瘀斑 |
| **气郁质** | 气机郁滞，神情抑郁 | 舌红，苔薄白 |
| **特禀质** | 先天失常，过敏体质 | 舌象多样，视过敏原 |

## 二、舌象与体质对应规律

### 2.1 舌质颜色与体质

| 舌质 | 对应体质 | 辨证要点 |
|------|----------|----------|
| 淡白 | 气虚、阳虚、血虚 | 气血不能上荣 |
| 淡红 | 平和质（正常） | 阴阳平衡 |
| 红 | 阴虚、湿热 | 热象 |
| 绛红 | 热入营血 | 重症热证 |
| 紫暗 | 血瘀、寒凝 | 气血运行不畅 |

### 2.2 舌苔与体质

| 舌苔 | 对应体质 | 辨证要点 |
|------|----------|----------|
| 薄白 | 平和质/外感初期 | 正常或表证 |
| 白厚 | 痰湿、寒湿、积食 | 邪气盛 |
| 黄苔 | 热证、湿热 | 热象 |
| 黄腻 | 湿热蕴结 | 热重于湿 |
| 少苔/无苔 | 阴虚、气阴两虚 | 阴液亏损 |
| 花剥 | 气阴两虚 | 部分胃气受损 |

### 2.3 舌形与体质

| 舌形 | 对应体质 | 辨证要点 |
|------|----------|----------|
| 胖大齿痕 | 气虚、阳虚、痰湿 | 水湿内停 |
| 瘦薄 | 阴虚、血虚 | 阴血不足 |
| 裂纹 | 阴虚、热盛、血虚 | 津液耗伤 |
| 点刺 | 热证、湿热 | 热邪亢盛 |
| 瘀斑瘀点 | 血瘀质 | 气血瘀滞 |

## 三、复合体质舌象识别

### 3.1 常见复合体质组合

| 组合类型 | 舌象特征 | 辨证要点 |
|----------|----------|----------|
| 气虚+痰湿 | 舌淡胖齿痕 + 苔厚腻 | 脾虚湿盛 |
| 阴虚+血瘀 | 舌红少苔 + 紫暗瘀点 | 阴虚血瘀 |
| 阳虚+血瘀 | 舌淡胖紫暗 + 苔白滑 | 寒凝血瘀 |
| 痰湿+湿热 | 舌胖苔厚腻黄 | 湿热蕴结 |
| 气郁+血瘀 | 舌暗红 + 瘀点 + 苔薄 | 肝郁血瘀 |

### 3.2 体质转化规律

```
阳虚 → 痰湿（阳虚水泛，湿聚成痰）
阴虚 → 血瘀（阴虚内热，热灼血液）
气虚 → 血瘀（气为血帅，气虚血瘀）
痰湿 → 湿热（郁久化热）
```

## 四、体质辨识问诊要点

### 4.1 平和质问诊

```
问：平时容易感冒吗？
→ 否 → 平和质可能性大
问：睡眠、饮食、二便都正常吗？
→ 是 → 平和质确认
```

### 4.2 气虚质问诊

```
特征：舌淡胖 + 齿痕 + 苔白
必问：
1. 容易疲劳吗？爬楼梯累吗？
2. 容易感冒吗？一年几次？
3. 说话声音低弱吗？
4. 出汗多吗？（自汗）
```

### 4.3 阳虚质问诊

```
特征：舌淡胖 + 苔白滑
必问：
1. 怕冷吗？手脚冰凉吗？
2. 吃凉东西不舒服吗？
3. 大便稀溏吗？
4. 腰膝酸冷吗？
```

### 4.4 阴虚质问诊

```
特征：舌红少苔 + 裂纹
必问：
1. 口干咽燥吗？喝水能缓解吗？
2. 手心脚心发热吗？
3. 晚上盗汗吗？
4. 大便干燥吗？
```

### 4.5 痰湿质问诊

```
特征：舌胖大 + 苔厚腻
必问：
1. 身体沉重吗？头昏蒙吗？
2. 胸闷吗？痰多吗？
3. 体型肥胖吗？腹部肥满？
4. 大便黏滞吗？
```

### 4.6 湿热质问诊

```
特征：舌红 + 苔黄腻
必问：
1. 脸上油光、长痘吗？
2. 口苦吗？小便黄吗？
3. 大便黏滞臭秽吗？
4. 情绪急躁吗？
```

### 4.7 血瘀质问诊

```
特征：舌紫暗 + 瘀点瘀斑
必问：
1. 身体哪里刺痛吗？
2. 疼痛固定吗？
3. 肤色晦暗吗？
4. 情绪抑郁吗？
```

### 4.8 气郁质问诊

```
特征：舌红 + 苔薄白
必问：
1. 情绪低落吗？爱叹气吗？
2. 胁肋胀痛吗？
3. 睡眠不好吗？
4. 消化不好吗？
```

## 五、体质调理配穴原则

### 5.1 九种体质配穴总纲

| 体质 | 治则 | 主穴 | 配穴 |
|------|------|------|------|
| 平和质 | 养生保健 | 足三里、三阴交 | 中脘、关元 |
| 气虚质 | 补中益气 | 足三里、气海 | 关元、百会 |
| 阳虚质 | 温补阳气 | 关元、命门 | 肾俞、灸法 |
| 阴虚质 | 滋阴清热 | 太溪、三阴交 | 照海、滋阴穴 |
| 痰湿质 | 化痰祛湿 | 阴陵泉、丰隆 | 中脘、脾俞 |
| 湿热质 | 清热利湿 | 曲池、阴陵泉 | 合谷、大椎 |
| 血瘀质 | 活血化瘀 | 血海、膈俞 | 太冲、三阴交 |
| 气郁质 | 疏肝解郁 | 太冲、肝俞 | 期门、神门 |
| 特禀质 | 疏风固表 | 曲池、血海 | 风池、肺俞 |

### 5.2 复合体质配穴

```
气虚+痰湿：
  → 健脾益气 + 化痰祛湿
  → 足三里 + 阴陵泉 + 丰隆 + 中脘

阴虚+血瘀：
  → 滋阴清热 + 活血化瘀
  → 太溪 + 三阴交 + 血海 + 膈俞

阳虚+血瘀：
  → 温补阳气 + 活血化瘀
  → 关元 + 命门 + 血海 + 膈俞（轻刺激）
```

## 六、舌象体质辨识流程

### 6.1 四步辨体质

```
Step 1：望舌色
  → 淡白 → 气虚/阳虚/血虚
  → 淡红 → 平和质
  → 红/绛 → 阴虚/热证
  → 紫暗 → 血瘀/寒凝

Step 2：望舌苔
  → 薄白 → 正常/表证
  → 白厚 → 痰湿/寒湿/积食
  → 黄苔 → 热证
  → 少/剥 → 阴虚

Step 3：望舌形
  → 胖大齿痕 → 气虚/阳虚/痰湿
  → 瘦薄 → 阴虚/血虚
  → 裂纹 → 阴虚/热盛
  → 瘀点 → 血瘀

Step 4：综合判定
  → 舌色 + 苔色 + 舌形 = 体质类型
  → 必要时问诊确认
```

### 6.2 舌象体质辨识速查表

| 舌象 | 舌苔 | 舌形 | 体质倾向 | 置信度 |
|------|------|------|----------|--------|
| 淡红 | 薄白 | 正常 | 平和质 | 高 |
| 淡白 | 薄白 | 胖大齿痕 | 气虚质 | 高 |
| 淡白 | 白滑 | 胖大齿痕 | 阳虚质 | 高 |
| 淡胖 | 白滑 | 胖大齿痕 | 阳虚+痰湿 | 中 |
| 红 | 少苔 | 裂纹 | 阴虚质 | 高 |
| 红 | 黄腻 | 胖大 | 湿热质 | 高 |
| 紫暗 | 少苔 | 瘀点 | 血瘀质 | 高 |
| 暗红 | 薄白 | 瘀点 | 气郁+血瘀 | 中 |

## 七、问诊问题生成规则（体质辨识专项）

### 7.1 舌象→体质问诊问题生成

```
舌象：淡白舌 + 齿痕
体质：气虚质/阳虚质
问诊：
  1. 你容易疲劳吗？（气虚）
  2. 你怕冷吗？手脚凉吗？（阳虚）

舌象：红舌 + 少苔 + 裂纹
体质：阴虚质
问诊：
  1. 你口干吗？喝水能缓解吗？
  2. 手心脚心热吗？
  3. 晚上盗汗吗？

舌象：舌胖大 + 苔厚腻
体质：痰湿质
问诊：
  1. 身体沉重吗？
  2. 痰多吗？
  3. 体型肥胖吗？

舌象：舌紫暗 + 瘀点
体质：血瘀质
问诊：
  1. 身体有固定刺痛吗？
  2. 皮肤容易青紫吗？
  3. 情绪抑郁吗？
```

### 7.2 十问歌体质版

```
一问寒热二问汗（怕冷/盗汗 → 阳虚/阴虚）
三问头身四问便（沉重/便溏 → 痰湿/气虚）
五问饮食六问胸（食欲/胸闷 → 脾胃/气郁）
七问聋渴八问眠（口渴/失眠 → 阴虚/血瘀）
九问旧病十问因（既往/诱因 → 辨证参考）
```

## 八、学习收获与规则提炼

### 8.1 核心认知

1. **体质是动态可调的**：舌象反映当前体质状态，可随调理变化
2. **复合体质很常见**：需综合分析，抓主要矛盾
3. **体质转化有规律**：阳虚→痰湿、阴虚→血瘀等
4. **问诊确认是关键**：舌象是参考，问诊是确认
