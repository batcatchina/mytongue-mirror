## 一、体质的稳定性与可变性

### 稳定性
体质是先天禀赋与后天长期生活习惯共同塑造的，具有相对稳定的特征。它构成了人体健康与疾病易感性的"底色"。

### 可变性
体质并非一成不变。通过饮食、运动、情志、环境及疾病过程的影响，体质可以发生动态变化。
- 长期过劳 → 气虚质向阳虚质转化
- 长期湿热环境 → 平和质转为湿热质

### 核心规律
体质是"慢变量"，证型是"快变量"。体质决定了证型发生的倾向性和演变路径，而证型是体质在特定诱因（外邪、内伤）下的急性或亚急性表达。

## 二、体质对证型倾向的影响

| 体质类型 | 常见证型（倾向性） | 典型舌象特征 |
| :--- | :--- | :--- |
| **平和质** | 表证（风寒、风热）、食积证 | 淡红舌，薄白苔 |
| **气虚质** | 气虚证、气陷证、血瘀证 | 舌淡胖，边有齿痕，苔薄白 |
| **阳虚质** | 阳虚证、寒湿证、水饮证 | 舌淡胖嫩，苔白滑或水滑 |
| **阴虚质** | 阴虚证、虚热证、燥证 | 舌红少苔或无苔，或有裂纹 |
| **痰湿质** | 痰湿证、湿阻证、痰浊证 | 舌淡胖，苔白厚腻或滑腻 |
| **湿热质** | 湿热证、肝胆湿热证、脾胃湿热证 | 舌红，苔黄厚腻或黄滑 |
| **血瘀质** | 血瘀证、气滞血瘀证、寒凝血瘀证 | 舌紫暗或有瘀点、瘀斑，舌下络脉曲张 |
| **气郁质** | 气滞证、肝郁证、痰气互结证 | 舌淡红或暗红，苔薄白或薄黄，舌边有瘀点 |
| **特禀质** | 风证、表证、过敏反应 | 舌象多变，常伴舌红、苔薄白或黄 |

## 三、证型在体质基础上的演变规律

### 从本到标
证型是体质偏颇的"标"象，是体质失衡的急性表现。

### 由浅入深
证型演变遵循"气→血→津液→脏腑"的层次：
- **气虚质**：气虚证 → 气陷证 → 血瘀证
- **阳虚质**：阳虚证 → 水饮证 → 寒凝经脉证

### 体质转化驱动证型转化
当体质发生根本性改变时，证型倾向也随之改变。

## 四、体质-证型辨证规则（条件→结论→行动）

### 规则1：气虚质基础证型识别
- **条件**：舌象【舌淡胖，边有齿痕，苔薄白】+ 主诉【乏力、气短、自汗】
- **结论**：倾向气虚证
- **行动**：健脾益气，推荐黄芪、党参、山药，避免过度劳累

### 规则2：阳虚质寒湿证演变预警
- **条件**：舌象【舌淡胖嫩，苔白滑】+ 主诉【畏寒、肢冷、便溏】
- **结论**：倾向阳虚寒湿证
- **行动**：温阳散寒化湿，推荐生姜、肉桂、茯苓，避免生冷寒凉

### 规则3：阴虚质虚热证识别
- **条件**：舌象【舌红少苔或无苔，有裂纹】+ 主诉【口干、盗汗、心烦】
- **结论**：倾向阴虚内热证
- **行动**：滋阴清热，推荐麦冬、沙参、百合，避免辛辣燥热

### 规则4：痰湿质痰浊证演变预警
- **条件**：舌象【舌淡胖，苔白厚腻或滑腻】+ 主诉【胸闷、痰多、体胖】
- **结论**：倾向痰湿内盛证
- **行动**：健脾化痰祛湿，推荐陈皮、茯苓、薏苡仁，控制高脂甜腻

### 规则5：湿热质湿热证识别
- **条件**：舌象【舌红，苔黄厚腻或黄滑】+ 主诉【口苦、大便黏滞、皮肤油腻】
- **结论**：倾向湿热蕴结证
- **行动**：清热利湿，推荐绿豆、冬瓜、赤小豆，避免辛辣油腻甜食

### 规则6：血瘀质血瘀证识别
- **条件**：舌象【舌紫暗或有瘀点、瘀斑，舌下络脉曲张】+ 主诉【刺痛、面色晦暗、唇色紫暗】
- **结论**：倾向血瘀证
- **行动**：活血化瘀，推荐山楂、玫瑰花、桃仁，避免寒凉久坐不动

### 规则7：气郁质气滞证识别
- **条件**：舌象【舌淡红或暗红，苔薄白，舌边有瘀点】+ 主诉【情绪抑郁、胸胁胀痛、叹气】
- **结论**：倾向肝郁气滞证
- **行动**：疏肝理气，推荐玫瑰花、佛手、薄荷，保持情志舒畅

## 五、P0级关键规则

### P0-1：体质转化预警规则 ⚠️
- **条件**：连续3次以上（间隔≥1个月）舌象从【A体质】向【B体质】明显转变
- **结论**：用户体质正在发生根本性改变
- **行动**：发出"体质转化预警"，提示调整生活方式，咨询专业中医师

### P0-2：证型与体质矛盾规则 ⚠️
- **条件**：当前证型与已知体质常见证型倾向完全相反（如：阳虚质出现湿热证）
- **结论**：可能存在"标本异象"或"体质转化"的复杂情况
- **行动**：输出"矛盾提示"，建议优先处理当前证型，关注体质变化

### P0-3：体质调理优先于证型治疗 ⚠️
- **条件**：长期（>3个月）反复出现同一类证型（如：反复发作的痰湿证）
- **结论**：核心问题在于体质偏颇未得到根本改善
- **行动**：引导用户从"治疗证型"转向"调理体质"，提供长期调理方案

### P0-4：避免医疗违规词 🚫
- 禁止："诊断"、"治疗"、"治愈"、"处方"、"用药"
- 替代："建议"、"推荐"、"倾向"、"提示"、"预警"、"咨询专业中医师"
