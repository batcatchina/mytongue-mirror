## 一、经脉循行与核心病候

### 1. 经脉循行路线
- **起点**：中焦
- **下行路线**：下络大肠，还循胃口，上膈，属肺
- **横出线路**：从肺系（气管、喉咙）横出腋下
- **下行路线**：下循臑内，行少阴心主之前，下肘中，循臂内上骨下廉，入寸口，上鱼，循鱼际，出大指之端
- **支脉**：从腕后（列缺）分出，走向食指桡侧端（与手阳明大肠经相接）

### 2. 是动则病（经脉异常表现）
- **肺胀满**：胸部胀满不适
- **膨膨而喘咳**：气喘、咳嗽有力（气盛）
- **缺盆中痛**：锁骨上窝疼痛
- **交两手而瞀**：双手交叉捂住胸部，视物模糊
- **臂厥**：手臂内侧发热、厥冷

### 3. 是主所生病（脏腑功能异常）
- **咳，上气喘渴**：咳嗽、气喘、口渴
- **烦心胸满**：心烦、胸闷
- **臑臂内前廉痛厥**：手臂内侧前缘疼痛、厥冷
- **掌中热**：掌心发热
- **气盛有余**：肩背痛、风寒、汗出中风、小便数而欠
- **气虚**：肩背痛寒、少气不足以息、溺色变

## 二、生理功能与辨证规则

### 1. 生理功能
- **主气司呼吸**：肺为"气之主"，主管呼吸运动，吸入清气，呼出浊气
- **主宣发肃降**：宣发向上向外布散卫气、津液；肃降向下向内清肃、通降
- **主通调水道**：通过宣发肃降调节水液代谢
- **朝百脉主治节**：全身血液汇聚于肺，辅助心脏调节气血运行

### 2. 辨证规则

#### 规则D43-1：肺气虚证
- **条件**：咳嗽无力 + 气短懒言 + 自汗畏风 + 易感冒 + 舌淡胖苔薄白 + 脉虚弱
- **结论**：肺气不足，宣降无力，卫外不固
- **行动**：补益肺气。取穴：太渊、足三里、肺俞、中府
- **⚠️ P0禁忌**：忌用泻法，手法宜轻柔、补法

#### 规则D43-2：肺阴虚证
- **条件**：干咳无痰 + 痰少而黏 + 口燥咽干 + 午后潮热 + 舌红少苔 + 脉细数
- **结论**：肺阴亏耗，虚热内扰，肺失清润
- **行动**：滋阴润肺。取穴：太溪、鱼际、肺俞、列缺
- **⚠️ P0禁忌**：忌用温燥、艾灸，手法宜轻柔、补法

#### 规则D43-3：肺热炽盛证
- **条件**：咳嗽气粗 + 痰黄稠量多 + 壮热口渴 + 咽喉肿痛 + 舌红苔黄燥 + 脉滑数
- **结论**：热邪壅肺，肺气郁闭，炼液为痰
- **行动**：清热泻肺。取穴：尺泽、少商（点刺放血）、鱼际、曲池
- **⚠️ P0禁忌**：忌用补法、艾灸，手法宜泻法

#### 规则D43-4：肺燥伤津证
- **条件**：干咳少痰 + 唇鼻干燥 + 咽喉干痛 + 大便干结 + 舌红少津苔薄黄 + 脉浮数
- **结论**：燥邪伤肺，肺失濡润，津液亏耗
- **行动**：清肺润燥。取穴：太渊、列缺、照海、三阴交
- **⚠️ P0禁忌**：忌用温燥、艾灸，手法宜轻柔、补法

#### 规则D43-5：痰湿阻肺证
- **条件**：咳嗽痰多 + 色白质稀 + 胸闷脘痞 + 纳呆呕恶 + 舌淡胖苔白腻 + 脉滑
- **结论**：脾失健运，水湿内停，聚湿成痰，阻于肺络
- **行动**：燥湿化痰，理气止咳。取穴：丰隆、尺泽、中府、阴陵泉
- **⚠️ P0禁忌**：忌用滋腻、补法，手法宜泻法

#### 规则D43-6：风寒束肺证
- **条件**：咳嗽声重 + 痰白清稀 + 鼻塞流清涕 + 恶寒发热无汗 + 舌淡红苔薄白 + 脉浮紧
- **结论**：风寒外束，肺气失宣
- **行动**：疏风散寒，宣肺止咳。取穴：列缺、风门、合谷、外关
- **⚠️ P0禁忌**：忌用寒凉、泻法，手法宜泻法，可配合艾灸

#### 规则D43-7：风热犯肺证
- **条件**：咳嗽频剧 + 痰黄黏稠 + 鼻塞流黄涕 + 咽喉红肿 + 舌尖红苔薄黄 + 脉浮数
- **结论**：风热犯表，肺失清肃
- **行动**：疏风清热，宣肺止咳。取穴：尺泽、鱼际、曲池、外关
- **⚠️ P0禁忌**：忌用温补、艾灸，手法宜泻法

## 七、禁忌与注意事项

### ⚠️ P0（安全禁忌）
- **孕妇禁忌**：禁用或少商、鱼际等井穴/荥穴的强刺激（放血、重按），以免引起子宫收缩；中府、云门等胸部穴位慎用，手法轻柔
- **手法禁忌**：
  - 肺气虚/肺阴虚：忌用泻法，手法宜轻柔、补法
  - 肺热炽盛/风热犯肺：忌用艾灸、温补，手法宜泻法
  - 皮肤感染、溃疡处禁止施术
  - 极度疲劳、饥饿、饱食、醉酒、精神高度紧张者不宜施术
- **常用禁忌**：
  - 尺泽、孔最等穴不宜过度使用，以免耗伤肺气
  - 太渊穴处避开桡动脉，避免损伤血管

### P1（重要）
- 少商点刺放血需严格消毒
- 咯血患者需排除肺结核、支气管扩张等器质性病变
- 哮喘急性发作需配合急救措施
