## 问诊补盲规则（问而确之）
CV识别存在天然盲区，推理时需自动标注"需问诊确认"的特征：
- **光线暗**→舌色可能偏暗偏深→问"你平时怕冷还是怕热？"区分真寒假寒
- **色差**→苔色白黄难辨→问"嘴里有没有发苦发黏？"辅助判断寒热
- **角度偏差**→舌形胖瘦不准→问"你觉得自己舌头偏胖还是偏瘦？"
- **齿痕浅**→可能漏识别→问"舌头边缘有没有牙印？"
- **分区模糊**→舌尖舌中混淆→问"你心烦吗？胃口怎么样？"定位心肺vs脾胃

推理输出应包含 confidence 和 needsConfirmation 字段，标注哪些结论需问诊验证。

## 问诊流程优先级
1. **年龄（必问第一项）**：决定整体推理框架，同一舌象25岁vs65岁解读完全不同
   - 60+：默认阳虚底子，寒湿必须补阳，防传变
   - 40-60：注意本虚标实，不可猛攻
   - 20-40：多实热，可清可泻
2. **主诉症状**：最不舒服的是什么？辅助定位脏腑
3. **二便情况**：大便成形否？小便频否？判断脾胃+肾
4. **睡眠情绪**：失眠？烦躁？判断心肝
5. **饮食口味**：口苦？口淡？纳差？判断脾胃+肝胆

## 问诊交互设计原则
- **后台复杂，前台简洁**：推理引擎做重活，用户只做选择
- **80%场景不问**：置信度>0.8直接出报告，零负担
- **20%场景问1-3题**：CV拿不准时才弹问题，选择题不打字
- **问题必须简单**：不是填问卷，是点按钮（"怕冷/怕热/都还行"）
- **年龄默认收集**：拍照前就让选年龄段（30以下/30-50/50-65/65以上），不觉得被问诊
- **渐进式深入**：只有用户主动点"想更准"才展开更多问诊

## 问诊问题生成规则（v3.2新增）

### 生成原则
- 问题必须简单通俗，用户看一眼就能答
- 选项2-3个，不要超过3个
- 优先问最能区分证型的问题
- 不要问专业术语（不问"你是不是脾虚"，问"胃口怎么样"）
- 每个问题都要附带简短的reason说明为什么问这个

### 场景问题模板
| 场景 | 问题 | 选项 |
|------|------|------|
| 寒热辨 | 你平时怕冷还是怕热？ | 怕冷/怕热/都还行 |
| 湿气辨 | 身体有没有沉重感？ | 经常有/偶尔/没有 |
| 虚实辨 | 容易累吗？ | 很容易/偶尔/精神还行 |
| 脾胃辨 | 胃口怎么样？ | 不太好/一般/挺好 |
| 肝郁辨 | 容易烦躁生气吗？ | 容易/偶尔/不容易 |
| 肾虚辨 | 腰膝酸软吗？ | 经常/偶尔/没有 |
| 心神辨 | 睡眠怎么样？ | 不好/一般/挺好 |
| 口味辨 | 嘴里发苦发黏吗？ | 发苦/发黏/都没有 |

### 修正推理规则
- 用户回答与初步推理一致：confidence提升0.1
- 用户回答与初步推理矛盾：该证型移除，重新推理
- 多个回答相互验证：confidence叠加，最高1.0

### 置信度阈值规则
| confidence | 行为 |
|------------|------|
| >= 0.8 | 不问诊，直接出报告 |
| 0.6-0.8 | 问1-2个选择题 |
| < 0.6 | 问2-3个选择题 |

### ⚠️ P0级：巨灵针灸—月经病辨证核心原则

来源：黄贵生老师师说（月经前头痛）

#### P0-月经-1：月经病三期辨证铁律
```
原则：女人受月经影响的毛病，不论精神和身体的毛病，只按月经病治。

三期辨证：
  1. 月经前出现的毛病 → 气滞（不论任何毛病，都按气滞治）
  2. 月经期间出现的毛病 → 瘀血（按瘀血治）
  3. 月经后出现的毛病 → 血虚（不论任何病按血虚治）

此为巨灵针灸治月经病原则：百治百灵，千治千灵。
```

条件：月经前2天左右头痛，眼睛疼肿，怕风

治法：神应针泻法，马上就好
注意：气滞用灸法、药法、按摩法都不好办，必须用针法泻气
