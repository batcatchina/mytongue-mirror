## 二、P0强制规则

### 规则1：年龄+证型双重约束
- **60岁以上患者 + 寒湿证型**
- 必须包含补阳穴位：关元、命门、肾俞
- 代码层 + prompt层双重保障

### 规则2：危急症状识别
发现以下症状，必须建议立即就医：
- 持续性胸痛
- 呼吸困难
- 意识模糊
- 剧烈头痛
- 大量出血

### 规则3：年龄分组建议
- 30岁以下：偏于清热、疏肝
- 30-50岁：平补平泻、调理脾胃
- 50-65岁：兼顾补虚、注意情志
- 65岁以上：必须顾护正气、补益为主

## 三、问诊问题生成规则

### 核心原则
1. 问题必须简单易懂，普通人能回答
2. 选项2-3个，不能太多
3. 每个问题都有明确的辨证意义
4. 问题顺序按辨证重要性排列

### 问题数量确定规则
| 置信度范围 | 问诊数量 |
|-----------|---------|
| >= 0.8 | 0个（不问）|
| 0.6 - 0.8 | 1-2个 |
| < 0.6 | 2-3个 |

### 问诊场景模板

#### 场景1：寒热辨（舌苔颜色不确定时）
```
问题：你平时怕冷还是怕热？
选项：["怕冷", "怕热", "都还行"]
原因：区分真寒假寒、热证
```

#### 场景2：湿气辨（苔质厚腻、胖大齿痕时）
```
问题：你感觉身体哪里不舒服？
选项：["头身困重", "消化不好", "小便异常", "都没有"]
原因：区分湿阻部位和程度
```

#### 场景3：虚实辨（舌体瘦薄或凹陷时）
```
问题：你平时容易疲劳吗？
选项：["很容易累", "偶尔疲劳", "精神挺好"]
原因：判断气血虚损程度
```

#### 场景4：气血辨（舌色淡白或淡红时）
```
问题：你的脸色看起来怎么样？
选项：["苍白/萎黄", "正常偏白", "红润"]
原因：判断血虚程度
```

#### 场景5：口苦口干辨
```
问题：你有口苦或口干的感觉吗？
选项：["经常口苦", "口干少津", "都没有"]
原因：区分肝胆湿热和阴虚
```

#### 场景6：睡眠辨
```
问题：你的睡眠质量怎么样？
选项：["很难入睡/易醒", "梦多", "睡眠尚可"]
原因：判断心神和阴液状况
```

#### 场景7：消化辨
```
问题：大便情况怎么样？
选项：["稀溏/便秘", "基本正常", "偶尔异常"]
原因：判断脾胃功能和湿邪
```

#### 场景8：情绪辨
```
问题：最近情绪怎么样？
```

## 四、证型规则库

### 4.1 湿热证
- **特征**：舌红 + 苔黄 + 苔腻/厚
- **治法**：清热祛湿
- **主穴**：足三里、阴陵泉、丰隆、曲池
- **辅穴**：内关、中脘

### 4.2 寒湿证
- **特征**：舌淡/淡紫 + 苔白 + 苔腻/厚
- **治法**：温阳祛湿
- **主穴**：关元、命门、肾俞、足三里、阴陵泉
- **注意**：60岁以上必须补阳

### 4.3 阴虚证
- **特征**：舌红 + 少苔/剥落 + 裂纹
- **治法**：滋阴清热
- **主穴**：三阴交、太溪、照海、肾俞
- **辅穴**：神门、安眠

### 4.4 气虚证
- **特征**：舌淡 + 苔薄 + 胖大/齿痕 + 凹陷
- **治法**：益气补虚
- **主穴**：气海、足三里、关元、肺俞
- **辅穴**：百会、中脘

### 4.5 血瘀证
- **特征**：舌紫/青紫 + 有瘀点/瘀斑
- **治法**：活血化瘀
- **主穴**：血海、三阴交、合谷、太冲
- **辅穴**：膈俞、肝俞

### 4.6 肝郁证
- **特征**：舌边红 + 苔薄 + 情绪异常
- **治法**：疏肝解郁
- **主穴**：太冲、肝俞、期门、膻中
- **辅穴**：内关、神门

### 4.7 痰湿证
- **特征**：舌胖大 + 苔厚腻 + 身体困重
- **治法**：化痰祛湿
- **主穴**：丰隆、阴陵泉、中脘、足三里
- **辅穴**：脾俞、胃俞

## 五、针灸配穴原则

### 5.1 区域配穴
| 区域 | 脏腑 | 主穴 |
|-----|------|------|
| 上焦（舌尖） | 心、肺、脑 | 膻中、肺俞、心俞、内关 |
| 中焦（舌中） | 脾、胃、肝、胆 | 中脘、足三里、阳陵泉、太冲 |
| 下焦（舌根） | 肾、膀胱、生殖 | 关元、肾俞、命门、三阴交 |

### 5.2 虚实配穴
| 病性 | 治法 | 配穴原则 |
|-----|------|---------|
| 虚证 | 补法 | 多用灸法、俞募穴 |
| 实证 | 泻法 | 多用针刺、远端穴 |
| 虚实夹杂 | 平补平泻 | 远近结合 |

### 5.3 年龄配穴
| 年龄段 | 配穴原则 |
|-------|---------|
| <30岁 | 清热为主，少用补法 |
| 30-50岁 | 平补平泻，调理脾胃 |
| >50岁 | 顾护正气，适当补益 |
| >60岁 | 必须补益，慎用泻法 |

## 六、问诊修正规则

### 6.1 回答影响逻辑

| 问题 | 回答 | 证型修正 |
|-----|------|---------|
| 怕冷/怕热 | 怕冷 | +寒证权重 |
| 怕冷/怕热 | 怕热 | +热证权重 |
| 疲劳程度 | 很容易累 | +气虚/血虚权重 |
| 脸色 | 苍白/萎黄 | +血虚权重 |
| 口苦 | 经常口苦 | +肝胆湿热权重 |
| 睡眠 | 难入睡/易醒 | +心神不宁权重 |

### 6.2 修正后的置信度计算
- 用户回答与初步辨证一致：置信度 + 0.1
- 用户回答与初步辨证矛盾：置信度 - 0.1，但需重新评估证型
- 用户回答不明确：置信度不变

## 四、靳三针辨证选穴规则（黄贵生老师）

> 来源：黄贵生老师靳三针课程笔记
> 整理原则：只提取"可操作的推理规则"——条件→结论→行动

### (一) 靳三针核心组方规则

#### 规则J01 | 智力低下/弱智儿童 | 智三针（神庭+本神双侧） + 手智针（劳宫+神门+内关） | P1
- 组成：神庭穴（前发际正中直上0.5寸）、本神穴（双侧，神庭旁开3寸）
- 手法：滞针法（快速捻针200次/分钟，捻到滞针后往后抽一下，再进针回去，抽5次）
- 配穴：可根据情况加足智针、舌三针
- 疗程：7次一个疗程，约5个疗程可见效

#### 规则J02 | 自闭症/孤独症 | 四神针 + 手智针 + 足智针，强刺激 | P1
- 四神针：百会穴前后左右各旁开1.5寸，针尖向中心对刺
- 手智针：劳宫、神门、内关（双侧）
- 足智针：涌泉、泉中、泉中内
- 注意：全部同侧，男左女右，隔天扎或每周三次
- 留针：30-45分钟，每5-10分钟行一次针

#### 规则J03 | 癫痫/痫证 | 痫三针（内关+申脉+照海）+ 智三针 | P1
- 组成：内关穴、申脉穴、照海穴
- 原理：申脉、照海是古派针法治癫痫要穴，加内关开心包
- 注意：老人刺内关不要过狠，避免耗费心包气

#### 规则J04 | 中风急救/预防后遗症 | 手指放血 + 颞三针（对侧）+ 太阴阳明大循环 | P0
- 颞三针：耳尖直上入发际2寸，及水平前后各旁开1寸
- 左瘫用右侧颞三针，右瘫用左侧颞三针
- 太阴阳明大循环=强心针法
- 关键：及时治疗，先机很重要

#### 规则J05 | 多动症/抽动症/眨眼 | 定神针 + 颞三针 + 手智针 | P1
- 定神针：印堂穴上0.5寸、阳白穴上0.5寸（双侧）
- 颞三针：耳尖直上入发际2寸
- 4岁小孩拼命眨眼睛：+ 合谷 + 陷谷
- 脖子后面膀胱经、胆经先拔罐松解
- 肝风跑到阳明经：在风池泻风

#### 规则J06 | 帕金森/共济失调 | 脑三针（脑户+脑空双侧） | P1
- 脑三针：脑户穴、脑空穴（双侧），直刺往下
- 原理：前病后治，脑袋对着前面的眼睛
- 对眼睛也有好处

### 一、经脉循行

**原文**：心主手厥阴心包络之脉，起于胸中，出属心包络，下膈，历络三焦；其支者，循胸出胁，下腋三寸，上抵腋下，循臑内，行太阴、少阴之间，入肘中，下臂，行两筋之间，入掌中，循中指，出其端；其支者，别掌中，循小指次指出其端。

**循行路线**：
1. 起于胸中，属于心包络，下膈历络三焦
2. 支脉从胸出胁，下腋三寸，上抵腋下（天池穴）
3. 沿上臂内侧，行太阴少阴之间，入肘中（曲泽穴）
4. 下臂行两筋之间，入掌中（劳宫穴），循中指出端（中冲穴）
5. 支脉从掌中别出，循无名指出端，与手少阳三焦经相接

### 三、是动病与主脉所生病

- **是动病**：手心热，臂肘挛急，腋肿，甚则胸胁支满，心中憺憺大动，面赤目黄，喜笑不休
- **主脉所生病**：烦心，心痛，掌中热

### 四、心包经病候辨证

#### 心包虚证
心悸怔忡，气短自汗，胸闷隐痛，面色淡白 → 补益心气养心安神 → 内关、神门、心俞、脾俞、足三里、气海

#### 心包实证
心胸憋闷刺痛，痛引肩背（瘀血）；或心烦失眠狂躁（痰火）→ 活血化瘀/清心豁痰 → 内关、郄门、膻中、膈俞、血海(瘀血)；大陵、劳宫、丰隆、中冲(痰火)

#### 心包寒证
心痛彻背，遇寒加重得温痛减，形寒肢冷 → 温通心阳散寒止痛 → 内关、心俞、厥阴俞、关元(灸)、命门(灸)

#### 心包热证
心中烦热，失眠多梦，口舌生疮，甚则神昏谵语 → 清心泻火凉血解毒 → 大陵、劳宫、中冲(刺血)、少府、曲泽

### 五、舌象特征

- 心火亢盛：舌尖红赤，或有芒刺溃疡，苔薄黄
- 心血瘀阻：舌质紫暗，或有瘀点瘀斑，舌下络脉紫暗迂曲
- 痰火扰心：舌质红，苔黄厚腻
- 心气(阳)虚：舌淡白或淡紫，舌体胖嫩边有齿痕，苔白滑
- 心阴(血)虚：舌红绛，少苔或无苔，或有裂纹

### 六、配穴方案

- 心痛心悸：内关+郄门+神门（虚配心俞足三里，实配膻中太冲）
- 失眠多梦：内关+神门+三阴交（配心俞肾俞安眠穴）

- 条件：气虚质（乏力、气短、自汗）持续存在，且出现口干、咽燥、手足心热、盗汗等阴虚表现。

- 行动：在益气基础上，增加养阴生津的调理思路。

#### 规则D34-2：气阴两虚向阴阳两虚演变
- 条件：气阴两虚证基础上，出现畏寒肢冷、腰膝酸软、夜尿频多等阳虚表现。
- 结论：证型由气阴两虚进一步演变为阴阳两虚。
- 行动：需阴阳双补，温阳与滋阴并用。

#### 规则D34-3：痰湿+血瘀兼夹体质证型叠加
- 条件：痰湿质（体胖、苔腻、脉滑）与血瘀质（面色晦暗、舌有瘀斑、脉涩）并存。
- 结论：证型表现为痰瘀互结，易出现胸痹、肢体麻木等。
- 行动：化痰祛湿与活血化瘀并行。

#### 规则D34-4：阳虚+气郁兼夹体质证型叠加
- 条件：阳虚质（畏寒、肢冷、喜暖）与气郁质（情绪抑郁、胁胀、善太息）并存。
- 结论：证型表现为阳虚气滞，易出现痛经、抑郁等。

#### 规则D34-5：小儿"纯阳之体"的证型转化
- 条件：小儿感受外邪后，易从阳化热。
- 结论：外感表证很快转化为里热证。

- 结论：原有气虚、血虚、气滞等证型在经期可能加重。

- 条件：孕期，阴血下聚以养胎。

- 行动：调理以养血安胎为主，慎用活血、峻下之品。

## 十、环境与情志因素

### P0规则

#### 规则D35-1：南方湿热舌象识别
- 条件：用户长期居住于南方（如广东、广西、海南），舌象表现为舌质偏红，舌苔黄腻、厚腻，舌体胖大边有齿痕。
- 结论：高度提示湿热质或痰湿质，易兼夹气郁。
- 行动：建议饮食清淡，减少甜腻、油炸、生冷食物摄入。可推荐赤小豆薏米水、冬瓜汤等。**孕妇忌用薏米，建议用茯苓、白术替代。**

#### 规则D35-2：熬夜致阴虚火旺舌象识别
- 条件：用户有长期熬夜史（超过1年，平均睡眠<6小时），舌象表现为舌质暗红或绛红，舌尖红赤，有瘀点，舌苔薄黄或剥苔。
- 结论：高度提示阴虚火旺、气阴两虚，易兼夹血瘀。
- 行动：强烈建议调整作息，保证夜间睡眠。可推荐酸枣仁、百合、麦冬泡水。**孕妇忌用酸枣仁，可用莲子心（少量）替代。**

### P1规则

#### 规则D35-3：北方干燥舌象识别
- 条件：用户长期居住于北方（如北京、内蒙、新疆），舌象表现为舌苔薄白或薄黄，干燥少津，舌质偏淡或偏红。
- 结论：提示津液亏耗或阴虚质倾向，易兼夹燥邪。
- 行动：建议增加饮水量，多食梨、银耳、百合等润燥食物。避免辛辣、烧烤。

#### 规则D35-4：吸烟致痰热瘀阻舌象识别
- 条件：用户有长期吸烟史（>10年，每日>10支），舌象表现为舌苔黄厚、腻，呈"烟苔"，舌质暗红或紫暗，舌下络脉粗张。
- 结论：高度提示痰热瘀阻、肺胃热盛。
- 行动：建议戒烟。饮食宜清淡，可推荐鱼腥草、罗汉果泡水。建议定期体检。

#### 规则D35-5：情志致肝气郁结舌象识别
- 条件：用户自述近期压力大、易怒或情绪低落，舌象表现为舌边红赤，点刺，舌苔薄黄或薄白。
- 结论：提示肝气郁结，若舌红苔黄则兼肝火。
- 行动：建议疏解情绪，可进行散步、瑜伽等运动。可推荐玫瑰花、佛手、代代花泡水。

#### 规则D35-6：思虑过度致心脾两虚舌象识别
- 条件：用户自述思虑过度、用脑频繁，舌象表现为舌质淡红或淡白，舌苔薄白或白腻，舌体胖大边有齿痕。
- 结论：提示心脾两虚、脾气虚。
- 行动：建议劳逸结合，避免过度用脑。可推荐龙眼肉、大枣、山药煮粥。

#### 规则D35-7：久坐致气滞血瘀舌象识别
- 条件：用户职业需久坐（如程序员、司机），舌象表现为舌质暗红或紫暗，舌下络脉粗张迂曲，舌苔白腻或黄腻。
- 结论：提示气滞血瘀，易兼夹痰湿。
- 行动：建议每1小时起身活动，进行拉伸。可推荐山楂、丹参泡水。**孕妇忌用丹参，可用玫瑰花、月季花替代。**

#### 规则D35-8：用脑过度致心肾不交舌象识别
- 条件：用户职业需高强度用脑（如科研人员、程序员），舌象表现为舌尖红赤，有瘀点，舌苔薄黄或剥苔，舌下络脉暗紫。
- 结论：提示心肾不交、阴虚火旺、心血瘀阻。
- 行动：建议保证睡眠，可进行冥想、深呼吸。可推荐黄连阿胶汤（需医师指导）或交泰丸（需医师指导）。**孕妇禁用黄连。**

#### 规则D35-9：惊恐致肾气不固舌象识别
- 条件：用户近期经历重大惊吓或长期处于恐惧状态，舌象表现为舌质淡白或青紫，舌苔白滑，舌体胖大或瘦小。
- 结论：提示肾气虚、肾阳虚，心神不宁。
- 行动：建议寻求心理支持，进行放松训练。可推荐核桃、芡实、莲子（去心）煮粥。

#### 规则D35-10：高原血瘀舌象识别
- 条件：用户长期居住或近期到访高原地区，舌象表现为舌质紫暗或暗红，舌下络脉粗张迂曲。
- 结论：提示血瘀质，与高原低氧环境相关。
- 行动：建议注意保暖，避免剧烈运动。可推荐红景天泡水。**孕妇禁用红景天，建议咨询医生。**

- **条件：** 舌质淡白或淡红，舌苔白腻或厚腻，舌体胖大。体形肥胖，胸闷痰多。
- **结论：** 痰湿质，需化痰祛湿。
- **行动：** 穴位：丰隆（主穴）、阴陵泉（配穴）。饮食忌甜腻、肥肉、糯米（生痰）。生活多运动（快走、游泳），少食多餐。**孕妇禁用大黄（泻下），慎用薏米（滑胎）。**

#### 规则D36-6：湿热质识别与调理
- **条件：** 舌质红，舌苔黄腻或厚腻，舌体可能胖大。面垢油光，易长痘。
- **结论：** 湿热质，需清热利湿。
- **行动：** 穴位：曲池（主穴）、阴陵泉（配穴）。饮食忌辛辣、油炸、酒（助湿生热）。生活保持通风，避免潮湿环境。**孕妇禁用大黄（泻下）、薏米（滑胎），慎用黄连（苦寒伤胎）。**

#### 规则D36-7：血瘀质识别与调理
- **条件：** 舌质紫暗或暗红，有瘀点或瘀斑，舌下络脉迂曲。肤色晦暗，易出现疼痛。

#### 规则D36-8：气郁质识别与调理
- **条件：** 舌质淡红或暗红，舌苔薄白或薄黄。情绪抑郁，胸胁胀闷。

### 规则D39-2：大肠津亏证

- **行动**：增液润燥，滑利润肠。配穴：天枢、支沟、照海、上巨虚

### 规则D39-3：大肠虚寒证

- **行动**：温阳散寒，涩肠止泻。配穴：神阙、天枢、关元、足三里

### 规则D39-4：大肠气滞证
- **条件**：舌淡红或暗红，苔薄白或薄黄，脉弦；伴有腹胀腹痛（痛无定处、排气后减轻）、便秘或便溏不爽、嗳气

## 十一、足阳明胃经辨证体系

### 辨证规则

### 规则D41-1：胃热证
- **条件**：舌红，苔黄厚或黄燥，脉滑数；伴有消谷善饥、口渴喜冷饮、口臭、牙龈肿痛、便秘、小便短赤
- **结论**：胃火炽盛，灼伤津液
- **行动**：清胃泻火，滋阴生津。配穴：内庭、厉兑、合谷、足三里（泻法）
- **⚠️ P0**：禁用温补、燥烈之品（如附子、肉桂、干姜）；禁用艾灸腹部及足三里

### 规则D41-2：胃阴虚证
- **条件**：舌红少苔或无苔，有裂纹，脉细数；伴有饥不欲食、胃脘隐痛或嘈杂、干呕、口干咽燥、大便干结
- **结论**：胃阴不足，虚热内生
- **行动**：滋养胃阴，降逆止呕。配穴：足三里（补法）、三阴交、太溪
- **⚠️ P0**：禁用苦寒直折之品（如黄连、黄芩、黄柏）；禁用泻法强刺激足三里

### 规则D41-3：食滞胃脘证
- **条件**：舌苔厚腻，色黄或白，脉滑实；伴有脘腹胀满、嗳腐吞酸、恶闻食臭、呕吐或泻下酸腐臭秽、矢气频作
- **结论**：饮食停滞，胃气不降
- **行动**：消食导滞，和胃降逆。配穴：中脘、天枢、足三里（泻法）、内关
- **⚠️ P0**：禁用补益、收涩之品（如人参、黄芪、五味子），以免闭门留寇

### 规则D41-4：胃气上逆证
- **条件**：舌苔白或腻，脉弦或滑；伴有呃逆、嗳气、恶心、呕吐、反酸
- **结论**：胃失和降，气逆于上
- **行动**：降逆和胃，理气止呕。配穴：中脘、内关、足三里（平补平泻）
- **⚠️ P0**：禁用升提、补气之品（如升麻、柴胡、黄芪）

### 规则D41-5：胃寒证（虚寒/实寒）
- **条件**：舌淡胖，苔白滑，脉沉迟或弦紧；伴有胃脘冷痛、得温痛减、遇寒加重、口淡不渴、呕吐清水、大便溏薄
- **结论**：寒邪客胃或胃阳不足
- **行动**：温胃散寒，行气止痛。配穴：中脘（温灸）、足三里（温灸）、神阙（隔姜灸）
- **⚠️ P0**：禁用苦寒清热之品（如石膏、知母、栀子）；禁用腹部及胃经穴的强刺激泻法
