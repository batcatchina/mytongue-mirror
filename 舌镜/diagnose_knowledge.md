## 问诊补盲规则（问而确之）
CV识别存在天然盲区，推理时需自动标注"需问诊确认"的特征：
- **光线暗**→舌色可能偏暗偏深→问"你平时怕冷还是怕热？"区分真寒假寒
- **色差**→苔色白黄难辨→问"嘴里有没有发苦发黏？"辅助判断寒热
- **角度偏差**→舌形胖瘦不准→问"你觉得自己舌头偏胖还是偏瘦？"
- **齿痕浅**→可能漏识别→问"舌头边缘有没有牙印？"
- **分区模糊**→舌尖舌中混淆→问"你心烦吗？胃口怎么样？"定位心肺vs脾胃

推理输出应包含 confidence 和 needsConfirmation 字段，标注哪些结论需问诊验证。

## 问诊流程优先级
1. **年龄（必问第一项）**：决定整体推理框架，同一舌象25岁vs65岁解读完全不同
   - 60+：默认阳虚底子，寒湿必须补阳，防传变
   - 40-60：注意本虚标实，不可猛攻
   - 20-40：多实热，可清可泻
2. **主诉症状**：最不舒服的是什么？辅助定位脏腑
3. **二便情况**：大便成形否？小便频否？判断脾胃+肾
4. **睡眠情绪**：失眠？烦躁？判断心肝
5. **饮食口味**：口苦？口淡？纳差？判断脾胃+肝胆

## 问诊交互设计原则
- **后台复杂，前台简洁**：推理引擎做重活，用户只做选择
- **80%场景不问**：置信度>0.8直接出报告，零负担
- **20%场景问1-3题**：CV拿不准时才弹问题，选择题不打字
- **问题必须简单**：不是填问卷，是点按钮（"怕冷/怕热/都还行"）
- **年龄默认收集**：拍照前就让选年龄段（30以下/30-50/50-65/65以上），不觉得被问诊
- **渐进式深入**：只有用户主动点"想更准"才展开更多问诊

## 问诊问题生成规则（v3.2新增）

### 生成原则
- 问题必须简单通俗，用户看一眼就能答
- 选项2-3个，不要超过3个
- 优先问最能区分证型的问题
- 不要问专业术语（不问"你是不是脾虚"，问"胃口怎么样"）
- 每个问题都要附带简短的reason说明为什么问这个

### 场景问题模板
| 场景 | 问题 | 选项 |
|------|------|------|
| 寒热辨 | 你平时怕冷还是怕热？ | 怕冷/怕热/都还行 |
| 湿气辨 | 身体有没有沉重感？ | 经常有/偶尔/没有 |
| 虚实辨 | 容易累吗？ | 很容易/偶尔/精神还行 |
| 脾胃辨 | 胃口怎么样？ | 不太好/一般/挺好 |
| 肝郁辨 | 容易烦躁生气吗？ | 容易/偶尔/不容易 |
| 肾虚辨 | 腰膝酸软吗？ | 经常/偶尔/没有 |
| 心神辨 | 睡眠怎么样？ | 不好/一般/挺好 |
| 口味辨 | 嘴里发苦发黏吗？ | 发苦/发黏/都没有 |

### 修正推理规则
- 用户回答与初步推理一致：confidence提升0.1
- 用户回答与初步推理矛盾：该证型移除，重新推理
- 多个回答相互验证：confidence叠加，最高1.0

### 置信度阈值规则
| confidence | 行为 |
|------------|------|
| >= 0.8 | 不问诊，直接出报告 |
| 0.6-0.8 | 问1-2个选择题 |
| < 0.6 | 问2-3个选择题 |

## 手少阳三焦经

### 一、经脉循行

**原文**：三焦手少阳之脉，起于小指次指之端，上出两指之间，循手表腕，出臂外两骨之间，上贯肘，循臑外，上肩，而交出足少阳之后，入缺盆，布膻中，散落心包，下膈，循属三焦；其支者，从膻中上出缺盆，上项系耳后，直上出耳上角，以屈下颊至頄，其支者，从耳后入耳中，出走耳前，过客主人前，交颊，至目锐眦。

**循行路线**：
1. **起始**：起于无名指末端（关冲穴）
2. **上肢**：沿无名指与中指之间上行，循手背腕部，出前臂外侧两骨之间，穿过肘尖，沿上臂外侧上达肩部

### 四、三焦腑病候辨证

#### 上焦病候（心肺）

- **虚证**：咳唾涎沫，面肿，遗尿，气短声低 → 补上焦益肺气 → 关元、太溪、肾俞、风门、肺俞
- **实证**：胸闷咳喘，胸中气噎不嗜食 → 泻实邪宣肺降气 → 关冲(泻)、尺泽、膻中、中府
- **寒证**：胸背相引而痛，嗌干津不纳，畏寒 → 温散上焦寒邪 → 风门、肺俞、大椎(灸)、列缺
- **热证**：口渴唇焦口臭鼻衄，咳唾脓血，舌干口焦咽闭 → 清泻上焦邪热 → 少商、商阳、尺泽、鱼际、内庭

#### 中焦病候（脾胃）

- **虚证**：脘腹胀满，食少纳呆，大便溏泄，神疲乏力 → 健运中焦补益脾胃 → 中脘、足三里、脾俞、胃俞、气海
- **实证**：脘腹胀痛，嗳腐吞酸，呕吐呃逆，便秘或下痢 → 通降中焦消食导滞 → 天枢、上巨虚、内关、支沟、中脘(泻)
- **寒证**：脘腹冷痛喜温，呕吐清水，口淡不渴 → 温中散寒 → 中脘(灸)、神阙(灸)、关元、足三里(灸)
- **热证**：胃脘灼痛，消谷善饥，口臭牙龈肿痛 → 清泻中焦胃热 → 内庭、厉兑、合谷、曲池、天枢

#### 下焦病候（肝肾）

- **虚证**：腰膝酸软，头晕耳鸣，遗精早泄，小便清长 → 补益下焦滋养肝肾 → 肾俞、太溪、关元、三阴交、肝俞
- **实证**：小腹胀痛，小便不利或癃闭，带下黄稠 → 清利下焦湿热 → 中极、膀胱俞、阴陵泉、三阴交、行间
- **寒证**：小腹冷痛畏寒肢冷，腰脊酸冷，夜尿多 → 温补下焦散寒 → 关元(灸)、命门(灸)、肾俞(灸)、太溪
- **热证**：小便短赤涩痛，或尿血便秘 → 清泻下焦湿热 → 中极、膀胱俞、委中、行间、侠溪

### 五、舌象特征

- 三焦气滞：舌淡红或暗红，苔薄白或薄黄，舌体偏胀或舌下络脉瘀曲
- 上焦热盛：舌尖红赤或有芒刺，苔薄黄或黄燥
- 中焦湿热：舌质红，苔黄厚腻，尤以舌中为甚
- 下焦湿热：舌根苔黄厚腻，或舌根红赤
- 三焦虚寒：舌淡白或淡紫，舌体胖嫩，苔白滑或白腻
- 三焦津亏：舌红绛，少苔或无苔，或苔干裂

## 手厥阴心包经

### 一、经脉循行

**原文**：心主手厥阴心包络之脉，起于胸中，出属心包络，下膈，历络三焦；其支者，循胸出胁，下腋三寸，上抵腋下，循臑内，行太阴、少阴之间，入肘中，下臂，行两筋之间，入掌中，循中指，出其端；其支者，别掌中，循小指次指出其端。

**循行路线**：
1. 起于胸中，属于心包络，下膈历络三焦
2. 支脉从胸出胁，下腋三寸，上抵腋下（天池穴）
3. 沿上臂内侧，行太阴少阴之间，入肘中（曲泽穴）
4. 下臂行两筋之间，入掌中（劳宫穴），循中指出端（中冲穴）
5. 支脉从掌中别出，循无名指出端，与手少阳三焦经相接

| 天泉 | 腋前纹头下2寸，肱二头肌长短头之间 | 心痛、胸胁胀满、咳嗽、胸背及上臂内侧痛 | 宽胸理气，通络止痛 |

### 三、是动病与主脉所生病

- **是动病**：手心热，臂肘挛急，腋肿，甚则胸胁支满，心中憺憺大动，面赤目黄，喜笑不休
- **主脉所生病**：烦心，心痛，掌中热

### 四、心包经病候辨证

#### 心包虚证
心悸怔忡，气短自汗，胸闷隐痛，面色淡白 → 补益心气养心安神 → 内关、神门、心俞、脾俞、足三里、气海

#### 心包实证
心胸憋闷刺痛，痛引肩背（瘀血）；或心烦失眠狂躁（痰火）→ 活血化瘀/清心豁痰 → 内关、郄门、膻中、膈俞、血海(瘀血)；大陵、劳宫、丰隆、中冲(痰火)

#### 心包寒证
心痛彻背，遇寒加重得温痛减，形寒肢冷 → 温通心阳散寒止痛 → 内关、心俞、厥阴俞、关元(灸)、命门(灸)

#### 心包热证
心中烦热，失眠多梦，口舌生疮，甚则神昏谵语 → 清心泻火凉血解毒 → 大陵、劳宫、中冲(刺血)、少府、曲泽

### 五、舌象特征

- 心血瘀阻：舌质紫暗，或有瘀点瘀斑，舌下络脉紫暗迂曲
- 痰火扰心：舌质红，苔黄厚腻
- 心气(阳)虚：舌淡白或淡紫，舌体胖嫩边有齿痕，苔白滑
- 心阴(血)虚：舌红绛，少苔或无苔，或有裂纹

- 失眠多梦：内关+神门+三阴交（配心俞肾俞安眠穴）

