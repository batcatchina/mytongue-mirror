## 二、P0强制规则

### 规则1：年龄+证型双重约束
- **60岁以上患者 + 寒湿证型**
- 必须包含补阳穴位：关元、命门、肾俞
- 代码层 + prompt层双重保障

### 规则2：危急症状识别
发现以下症状，必须建议立即就医：
- 持续性胸痛
- 呼吸困难
- 意识模糊
- 剧烈头痛
- 大量出血

### 规则3：年龄分组建议
- 30岁以下：偏于清热、疏肝
- 30-50岁：平补平泻、调理脾胃
- 50-65岁：兼顾补虚、注意情志
- 65岁以上：必须顾护正气、补益为主

## 三、问诊问题生成规则

### 核心原则
1. 问题必须简单易懂，普通人能回答
2. 选项2-3个，不能太多
3. 每个问题都有明确的辨证意义
4. 问题顺序按辨证重要性排列

### 问题数量确定规则
| 置信度范围 | 问诊数量 |
|-----------|---------|
| >= 0.8 | 0个（不问）|
| 0.6 - 0.8 | 1-2个 |
| < 0.6 | 2-3个 |

### 问诊场景模板

#### 场景1：寒热辨（舌苔颜色不确定时）
```
问题：你平时怕冷还是怕热？
选项：["怕冷", "怕热", "都还行"]
原因：区分真寒假寒、热证
```

#### 场景2：湿气辨（苔质厚腻、胖大齿痕时）
```
问题：你感觉身体哪里不舒服？
选项：["头身困重", "消化不好", "小便异常", "都没有"]
原因：区分湿阻部位和程度
```

#### 场景3：虚实辨（舌体瘦薄或凹陷时）
```
问题：你平时容易疲劳吗？
选项：["很容易累", "偶尔疲劳", "精神挺好"]
原因：判断气血虚损程度
```

#### 场景4：气血辨（舌色淡白或淡红时）
```
问题：你的脸色看起来怎么样？
选项：["苍白/萎黄", "正常偏白", "红润"]
原因：判断血虚程度
```

#### 场景5：口苦口干辨
```
问题：你有口苦或口干的感觉吗？
选项：["经常口苦", "口干少津", "都没有"]
原因：区分肝胆湿热和阴虚
```

#### 场景6：睡眠辨
```
问题：你的睡眠质量怎么样？
选项：["很难入睡/易醒", "梦多", "睡眠尚可"]
原因：判断心神和阴液状况
```

#### 场景7：消化辨
```
问题：大便情况怎么样？
选项：["稀溏/便秘", "基本正常", "偶尔异常"]
原因：判断脾胃功能和湿邪
```

#### 场景8：情绪辨
```
问题：最近情绪怎么样？
```

## 四、证型规则库

### 4.1 湿热证
- **特征**：舌红 + 苔黄 + 苔腻/厚
- **治法**：清热祛湿
- **主穴**：足三里、阴陵泉、丰隆、曲池
- **辅穴**：内关、中脘

### 4.2 寒湿证
- **特征**：舌淡/淡紫 + 苔白 + 苔腻/厚
- **治法**：温阳祛湿
- **主穴**：关元、命门、肾俞、足三里、阴陵泉
- **注意**：60岁以上必须补阳

### 4.3 阴虚证
- **特征**：舌红 + 少苔/剥落 + 裂纹
- **治法**：滋阴清热
- **主穴**：三阴交、太溪、照海、肾俞
- **辅穴**：神门、安眠

### 4.4 气虚证
- **特征**：舌淡 + 苔薄 + 胖大/齿痕 + 凹陷
- **治法**：益气补虚
- **主穴**：气海、足三里、关元、肺俞
- **辅穴**：百会、中脘

### 4.5 血瘀证
- **特征**：舌紫/青紫 + 有瘀点/瘀斑
- **治法**：活血化瘀
- **主穴**：血海、三阴交、合谷、太冲
- **辅穴**：膈俞、肝俞

### 4.6 肝郁证
- **特征**：舌边红 + 苔薄 + 情绪异常
- **治法**：疏肝解郁
- **主穴**：太冲、肝俞、期门、膻中
- **辅穴**：内关、神门

### 4.7 痰湿证
- **特征**：舌胖大 + 苔厚腻 + 身体困重
- **治法**：化痰祛湿
- **主穴**：丰隆、阴陵泉、中脘、足三里
- **辅穴**：脾俞、胃俞

## 五、针灸配穴原则

### 5.1 区域配穴
| 区域 | 脏腑 | 主穴 |
|-----|------|------|
| 上焦（舌尖） | 心、肺、脑 | 膻中、肺俞、心俞、内关 |
| 中焦（舌中） | 脾、胃、肝、胆 | 中脘、足三里、阳陵泉、太冲 |
| 下焦（舌根） | 肾、膀胱、生殖 | 关元、肾俞、命门、三阴交 |

### 5.2 虚实配穴
| 病性 | 治法 | 配穴原则 |
|-----|------|---------|
| 虚证 | 补法 | 多用灸法、俞募穴 |
| 实证 | 泻法 | 多用针刺、远端穴 |
| 虚实夹杂 | 平补平泻 | 远近结合 |

### 5.3 年龄配穴
| 年龄段 | 配穴原则 |
|-------|---------|
| <30岁 | 清热为主，少用补法 |
| 30-50岁 | 平补平泻，调理脾胃 |
| >50岁 | 顾护正气，适当补益 |
| >60岁 | 必须补益，慎用泻法 |

## 六、问诊修正规则

### 6.1 回答影响逻辑

| 问题 | 回答 | 证型修正 |
|-----|------|---------|
| 怕冷/怕热 | 怕冷 | +寒证权重 |
| 怕冷/怕热 | 怕热 | +热证权重 |
| 疲劳程度 | 很容易累 | +气虚/血虚权重 |
| 脸色 | 苍白/萎黄 | +血虚权重 |
| 口苦 | 经常口苦 | +肝胆湿热权重 |
| 睡眠 | 难入睡/易醒 | +心神不宁权重 |

### 6.2 修正后的置信度计算
- 用户回答与初步辨证一致：置信度 + 0.1
- 用户回答与初步辨证矛盾：置信度 - 0.1，但需重新评估证型
- 用户回答不明确：置信度不变

## 四、靳三针辨证选穴规则（黄贵生老师）

> 来源：黄贵生老师靳三针课程笔记
> 整理原则：只提取"可操作的推理规则"——条件→结论→行动

### (一) 靳三针核心组方规则

#### 规则J01 | 智力低下/弱智儿童 | 智三针（神庭+本神双侧） + 手智针（劳宫+神门+内关） | P1
- 组成：神庭穴（前发际正中直上0.5寸）、本神穴（双侧，神庭旁开3寸）
- 手法：滞针法（快速捻针200次/分钟，捻到滞针后往后抽一下，再进针回去，抽5次）
- 配穴：可根据情况加足智针、舌三针
- 疗程：7次一个疗程，约5个疗程可见效

#### 规则J02 | 自闭症/孤独症 | 四神针 + 手智针 + 足智针，强刺激 | P1
- 四神针：百会穴前后左右各旁开1.5寸，针尖向中心对刺
- 手智针：劳宫、神门、内关（双侧）
- 足智针：涌泉、泉中、泉中内
- 注意：全部同侧，男左女右，隔天扎或每周三次
- 留针：30-45分钟，每5-10分钟行一次针

#### 规则J03 | 癫痫/痫证 | 痫三针（内关+申脉+照海）+ 智三针 | P1
- 组成：内关穴、申脉穴、照海穴
- 原理：申脉、照海是古派针法治癫痫要穴，加内关开心包
- 注意：老人刺内关不要过狠，避免耗费心包气

#### 规则J04 | 中风急救/预防后遗症 | 手指放血 + 颞三针（对侧）+ 太阴阳明大循环 | P0
- 颞三针：耳尖直上入发际2寸，及水平前后各旁开1寸
- 左瘫用右侧颞三针，右瘫用左侧颞三针
- 太阴阳明大循环=强心针法
- 关键：及时治疗，先机很重要

#### 规则J05 | 多动症/抽动症/眨眼 | 定神针 + 颞三针 + 手智针 | P1
- 定神针：印堂穴上0.5寸、阳白穴上0.5寸（双侧）
- 颞三针：耳尖直上入发际2寸
- 4岁小孩拼命眨眼睛：+ 合谷 + 陷谷
- 脖子后面膀胱经、胆经先拔罐松解
- 肝风跑到阳明经：在风池泻风

#### 规则J06 | 帕金森/共济失调 | 脑三针（脑户+脑空双侧） | P1
- 脑三针：脑户穴、脑空穴（双侧），直刺往下
- 原理：前病后治，脑袋对着前面的眼睛
- 对眼睛也有好处

### 一、经脉循行

**原文**：心主手厥阴心包络之脉，起于胸中，出属心包络，下膈，历络三焦；其支者，循胸出胁，下腋三寸，上抵腋下，循臑内，行太阴、少阴之间，入肘中，下臂，行两筋之间，入掌中，循中指，出其端；其支者，别掌中，循小指次指出其端。

**循行路线**：
1. 起于胸中，属于心包络，下膈历络三焦
2. 支脉从胸出胁，下腋三寸，上抵腋下（天池穴）
3. 沿上臂内侧，行太阴少阴之间，入肘中（曲泽穴）
4. 下臂行两筋之间，入掌中（劳宫穴），循中指出端（中冲穴）
5. 支脉从掌中别出，循无名指出端，与手少阳三焦经相接

### 三、是动病与主脉所生病

- **是动病**：手心热，臂肘挛急，腋肿，甚则胸胁支满，心中憺憺大动，面赤目黄，喜笑不休
- **主脉所生病**：烦心，心痛，掌中热

### 四、心包经病候辨证

#### 心包虚证
心悸怔忡，气短自汗，胸闷隐痛，面色淡白 → 补益心气养心安神 → 内关、神门、心俞、脾俞、足三里、气海

#### 心包实证
心胸憋闷刺痛，痛引肩背（瘀血）；或心烦失眠狂躁（痰火）→ 活血化瘀/清心豁痰 → 内关、郄门、膻中、膈俞、血海(瘀血)；大陵、劳宫、丰隆、中冲(痰火)

#### 心包寒证
心痛彻背，遇寒加重得温痛减，形寒肢冷 → 温通心阳散寒止痛 → 内关、心俞、厥阴俞、关元(灸)、命门(灸)

#### 心包热证
心中烦热，失眠多梦，口舌生疮，甚则神昏谵语 → 清心泻火凉血解毒 → 大陵、劳宫、中冲(刺血)、少府、曲泽

### 五、舌象特征

平衡阴阳，调和气血

### 饮食调养
- **宜**：均衡饮食，顺应四时，春多辛散，夏多清凉，秋多滋润，冬多温补
- **忌**：暴饮暴食，偏嗜五味，过寒过热

### 穴位配伍
- **主穴**：足三里
- **配穴**：涌泉、百会
- **操作**：每日按揉或艾灸，每穴3-5分钟

## 4.2 气虚质调理方案

- **忌**：生冷、油腻、辛辣耗气之品（萝卜生食、山楂、槟榔、浓茶、咖啡）

### 穴位配伍

## 4.3 阳虚质调理方案

- **忌**：⚠️ **生冷寒凉**：冰镇饮料、西瓜、梨、苦瓜、黄瓜、螃蟹、鸭肉

## 4.4 阴虚质调理方案

### 饮食调养

- **忌**：⚠️ **辛辣燥热**：辣椒、花椒、葱蒜、韭菜、羊肉、狗肉、煎炸食物

## 4.5 痰湿质调理方案

## 4.9 特禀质调理方案

- 🔴 **孕妇禁忌**：避免活血化瘀、峻下逐水、有毒中药，需专业医师指导
