# 经络拍打机器人 - 产品设计方案

> 定位：舌镜AI辨证的执行端，让辨证结果落地
> 核心思路：大范围拍打经络，不需要精准穴位定位

---

## 一、产品定位

**名称**：经络拍手 / 智能经络拍打仪

**用户场景**：
- 舌镜照出经络淤堵 → 输出调理方案 → 机器手执行拍打
- 用户躺着/坐着，机器手自动沿经络走向移动拍打

**核心价值**：
- 舌镜"照得准"，拍手"打得着"，形成检测→调理闭环
- 软硬结合，壁垒深厚

---

## 二、技术方案

### 2.1 动力系统（基于筋膜枪改装）

现有筋膜枪参数：
- 振动频率：1500-3500 RPM
- 电机：无刷直流电机（BLDC）
- 冲击力：10-30kg可调

改装要点：
- 增加**可控变频模块**，支持1-10档力度
- 增加**定时功能**，按辨证方案执行指定时长
- 保留原有手柄，用户可手持，也可装在导轨上

### 2.2 移动机构

**方案A：手持式（最简）**
- 用户手持筋膜枪，沿经络走向移动
- 舌镜APP实时显示当前拍打经络位置
- 成本最低，改动最小

**方案B：导轨式（推荐）**
- 拍打头装在滑轨上，可沿着经络走向移动
- 步进电机驱动，位置精度约1cm
- 用户躺下，机器手沿身体移动

**方案C：多轴机械臂（长期）**
- 6轴以下机械臂，模拟人手拍打轨迹
- 成本高，结构复杂

### 2.3 拍打头设计

核心参数：
- 拍打头直径：5-8cm（覆盖整条经络）
- 材质：硅胶+砭石（中医元素）
- 形状：圆形或椭圆形，接触面积大

可选配置：
- 加热功能（艾灸效果）
- 负压吸附（拔罐+拍打二合一）

## 三、规格参数

| 参数 | 数值 |
|------|------|
| 力度档位 | 1-10级（通过PWM调速） |
| 频率范围 | 1500-3500 RPM |
| 拍打头直径 | 5-8cm |
| 移动速度 | 0-10cm/s（可调） |
| 单次时长 | 1-30分钟（可调） |
| 供电 | 锂电池，续航2小时 |
| 通讯 | 蓝牙5.0 / WiFi |
| 控制方式 | 舌镜APP控制，或手持独立使用 |

## 四、与舌镜联动流程

```
舌镜辨证 → 生成调理方案
    ↓
推送经络信息（哪条经、哪个部位）
    ↓
经络拍手机接收指令
    ↓
自动移动到对应经络位置
    ↓
执行拍打（力度+频次+时长）
    ↓
实时显示拍打部位
    ↓
调理完成，数据回传舌镜
```

### 舌镜APP显示内容

- 当前拍打经络名称（如：足太阳膀胱经）
- 拍打进度（已打/总时长）
- 力度和频次
- 预计剩余时间

## 六、软件接口

### 指令协议（JSON格式）

```json
{
  "action": "start",
  "channel": "足太阳膀胱经",
  "position": "背部脊柱两侧",
  "intensity": 7,       // 力度1-10
  "frequency": 2500,    // RPM
  "duration": 300,      // 秒
  "method": "back_and_forth"  // 来回拍打
}
```

### 响应协议

```json
{
  "status": "running",
  "channel": "足太阳膀胱经",
  "elapsed": 60,
  "remaining": 240
}
```

## 七、产品形态建议

### 短期（可快速落地）
- **手持筋膜枪改装版**
- 用户自己购买筋膜枪+APP控制
- 舌镜输出经络位置，用户手持沿经络移动拍打
- 改动最小，验证市场

### 中期（产品化）
- **一体化经络拍打仪**
- 导轨式设计，机器手沿导轨移动
- 用户躺下，设备自动沿经络拍打
- 增加位置传感器，实时反馈位置

### 长期（高端）
- **多阵列拍打机器人**
- 多个拍打头同时工作
- 可同时处理多条经络
- 增加加热+负压+艾灸功能

---

## 八、关键挑战

1. **供应链**：需要找筋膜枪/按摩仪工厂合作定制
2. **软硬协议**：定义舌镜与机器手的通讯协议
3. **安全性**：拍打力度过大会伤到软组织，需要限位保护
4. **认证**：健康类电子产品需要相关认证（FDA/CFDA）
