概述

舌镜 API 为第三方应用提供舌象识别与中医辨证判断能力。基于 四层推理辨证引擎 和 500+ 中医经典规则， 提供舌象识别、证型判定、配穴方案、调护建议等完整服务。

💡
产品定位
舌镜是辨证能力引擎，不是舌诊 app。为 Agent 提供即插即用的辨证能力——谁需要辨证能力，接上来就用。

认证

所有 API 请求需要通过 API Key 进行认证。请将您的 API Key 放在 HTTP 请求头的 X-API-Key 字段中。

请求示例

curl -X POST https://api.shezhen.top/v1/tongue-ai/analyze \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your_api_key_here" \
  -d '{
    "image": "data:image/jpeg;base64,/9j/4AAQ..."
  }'
⚠️
安全提示
请勿将 API Key 直接暴露在前端代码中。建议通过后端服务转发 API 请求。

POST /tongue-ai/analyze 舌象识别与辨证分析

舌象识别与辨证分析主接口。上传舌象图片，返回完整的舌诊分析结果、辨证结论、配穴方案和调护建议。

请求头

字段 类型 必填 说明
Content-Type string 是 application/json
X-API-Key string 是 API 密钥
X-App-Id string 否 应用标识（用于渠道统计）

  "image": "data:image/jpeg;base64,/9j/4AAQ...",  // 图片数据（Base64 或 URL）

MCP 协议接入

舌镜 MCP Server 提供标准化的 Model Context Protocol 接口，支持 AI Agent 通过 MCP 协议直接调用舌象辨证能力。

服务器信息

配置项 值
Server URL https://mcp.zheng-he.top/mcp
协议版本 MCP v1.0
认证方式 API Key (Header)

可用工具

1. tongue_analyze → 2. tongue_interrogate → 3. acupoint_query → 4. constitution_identify → 5. health_profile
工具名称 功能 参数
tongue_analyze 舌象识别与辨证分析 image (string), patient_info? (object)
tongue_interrogate 问诊交互 context (object), user_response? (string)
acupoint_query 穴位信息查询 name (string)
constitution_identify 中医体质辨识 tongue_features (object)
health_profile 生成健康画像 syndrome_result (object)

调用流程

// 1. 初始化 MCP Client
import { MCPClient } from '@modelcontextprotocol/sdk';

const client = new MCPClient({
  serverUrl: 'https://mcp.zheng-he.top/mcp',
  apiKey: 'your_api_key'
});

// 2. 调用舌象分析工具
const result = await client.callTool('tongue_analyze', {
  image: 'data:image/jpeg;base64,...',
  patient_info: { age: 35, gender: 'female' }
});

// 3. 获取结果
console.log(result.data.syndrome_differentiation);

SDK & 示例

TypeScript SDK

npm install tongue-mirror-sdk
import { TongueMirror } from 'tongue-mirror-sdk';

const client = new TongueMirror({
  apiKey: 'your_api_key'
});

// 舌象分析
const result = await client.analyze({
  image: 'data:image/jpeg;base64,...',
  patientInfo: { age: 35, gender: 'female' }
});

if (result.success) {
  console.log(`证型: ${result.data.syndrome_differentiation.primary.syndrome}`);
  console.log(`置信度: ${result.data.syndrome_differentiation.primary.confidence}`);
}

Python SDK

pip install tongue-mirror-sdk
from tongue_mirror import TongueMirror

client = TongueMirror(api_key="your_api_key")

result = client.analyze(
    image="data:image/jpeg;base64,...",
    patient_info={"age": 35, "gender": "female"}
)

if result["success"]:
    syndrome = result["data"]["syndrome_differentiation"]["primary"]["syndrome"]
    print(f"证型: {syndrome}")
