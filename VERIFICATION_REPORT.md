## 四、规则匹配引擎设计

### 4.1 匹配优先级

| 优先级 | 规则类型 | 权重范围 | 说明 |
|--------|----------|----------|------|
| 1 | 三维综合规则 | 70-85 | 3个维度都匹配，最高权重 |
| 2 | 舌态+舌质复合 | 50-60 | 危重舌象优先匹配 |
| 3 | 舌质+舌苔复合 | 42-60 | 2个维度匹配 |
| 4 | 舌形+舌苔复合 | 48-55 | 2个维度匹配 |
| 5 | 单特征规则 | 10-55 | 基础权重 |

### 4.2 匹配算法

```
1. 遍历所有规则
2. 检查规则条件是否完全匹配
3. 计算匹配得分 = 匹配条件数/总条件数 × 规则权重
4. 舌态危急优先加分（强硬/痿软/歪斜 +10分）
5. 按得分降序排序
6. 返回Top3结果
```

### 8.2 辨证优先级规则

1. **舌态危急优先**: 短缩、强硬属危重，急则治标
2. **舌色定寒热**: 舌色是判断寒热最可靠指标
3. **舌苔定表里**: 薄白为表，厚腻为里
4. **舌形定虚实**: 胖大苔厚为实，瘦薄少苔为虚
5. **复合特征综合**: 多特征矛盾时，以权重最高者为主
